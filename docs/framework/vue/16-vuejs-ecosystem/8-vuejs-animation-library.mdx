# Vue.js动画库

在现代 Web 开发中，动画是提升用户体验的重要工具。Vue.js 提供了强大的内置动画支持，同时也兼容许多流行的第三方动画库。本文将带你了解 Vue.js 动画的基础知识，并通过实际案例展示如何为你的应用添加动画效果。

## 什么是 Vue.js 动画？

Vue.js 动画是通过 Vue 的内置过渡系统实现的。它允许你在元素插入、更新或从 DOM 中移除时，应用 CSS 过渡或 JavaScript 动画。Vue 的过渡系统非常灵活，可以轻松地与 CSS 动画库（如 Animate.css）或 JavaScript 动画库（如 GSAP）集成。

### Vue.js内置动画

Vue.js 提供了 `<transition>` 和 `<transition-group>` 组件，用于处理单个元素或列表的过渡效果。以下是一个简单的例子，展示如何使用 `<transition>` 组件实现淡入淡出效果：

```vue
<template>
  <div>
    <button @click="show = !show">Toggle</button>
    <transition name="fade">
      <p v-if="show">Hello, Vue.js!</p>
    </transition>
  </div>
</template>

<script>
export default {
  data() {
    return {
      show: true
    };
  }
};
</script>

<style>
.fade-enter-active, .fade-leave-active {
  transition: opacity 0.5s;
}
.fade-enter, .fade-leave-to {
  opacity: 0;
}
</style>
```

在这个例子中，当 `show` 的值发生变化时，`<p>` 元素会以淡入淡出的方式显示或隐藏。`fade-enter-active` 和 `fade-leave-active` 类定义了过渡的持续时间，而 `fade-enter` 和 `fade-leave-to` 类定义了过渡的起始和结束状态。

### 使用第三方动画库

除了内置的过渡系统，Vue.js 还可以与许多第三方动画库集成。例如，Animate.css 是一个流行的 CSS 动画库，提供了丰富的预定义动画效果。以下是如何在 Vue.js 中使用 Animate.css 的示例：

```vue
<template>
  <div>
    <button @click="show = !show">Toggle</button>
    <transition
      enter-active-class="animate__animated animate__bounceIn"
      leave-active-class="animate__animated animate__bounceOut"
    >
      <p v-if="show">Hello, Animate.css!</p>
    </transition>
  </div>
</template>

<script>
export default {
  data() {
    return {
      show: true
    };
  }
};
</script>

<style>
@import 'https://cdnjs.cloudflare.com/ajax/libs/animate.css/4.1.1/animate.min.css';
</style>
```

在这个例子中，我们使用了 Animate.css 提供的 `bounceIn` 和 `bounceOut` 动画效果。通过 `enter-active-class` 和 `leave-active-class` 属性，我们可以指定元素进入和离开时的动画类。

## 实际应用场景

### 列表排序动画

在实际应用中，列表排序动画是一个常见的需求。Vue.js 的 `<transition-group>` 组件可以轻松实现这一效果。以下是一个简单的例子，展示如何为列表项添加排序动画：

```vue
<template>
  <div>
    <button @click="shuffle">Shuffle</button>
    <transition-group name="list" tag="ul">
      <li v-for="item in items" :key="item" class="list-item">
        {{ item }}
      </li>
    </transition-group>
  </div>
</template>

<script>
export default {
  data() {
    return {
      items: [1, 2, 3, 4, 5]
    };
  },
  methods: {
    shuffle() {
      this.items = this.items.sort(() => Math.random() - 0.5);
    }
  }
};
</script>

<style>
.list-item {
  transition: all 1s;
}
.list-enter, .list-leave-to {
  opacity: 0;
  transform: translateY(30px);
}
.list-leave-active {
  position: absolute;
}
</style>
```

在这个例子中，当点击 "Shuffle" 按钮时，列表项会随机排序，并且每个项都会以动画的形式移动到新的位置。
