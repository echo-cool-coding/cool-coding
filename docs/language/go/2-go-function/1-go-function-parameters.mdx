# Go 函数参数

在Go语言中，函数是代码的基本构建块之一。函数参数是函数定义中用于接收外部数据的变量。通过参数，我们可以将数据传递给函数，并在函数内部进行处理。本文将详细介绍Go函数参数的概念、使用方法以及实际应用场景。

## 什么是函数参数？

函数参数是函数定义中声明的变量，用于接收调用函数时传递的值。参数允许我们将数据从函数外部传递到函数内部，从而实现更灵活和可重用的代码。

### 基本语法

在Go中，函数参数的定义位于函数名之后的括号内。每个参数由参数名和参数类型组成，多个参数之间用逗号分隔。

```go
func 函数名(参数1 类型1, 参数2 类型2) 返回类型 {
    // 函数体
}
```

例如，定义一个简单的函数 `add`，它接收两个整数参数并返回它们的和：

```go
func add(a int, b int) int {
    return a + b
}
```

### 调用函数并传递参数

调用函数时，需要按照函数定义的参数顺序传递相应的值。例如：

```go
result := add(3, 5)
fmt.Println(result) // 输出: 8
```

## 参数的类型

Go语言中的函数参数可以是任何有效的类型，包括基本类型（如 `int`、`string`）、复合类型（如 `slice`、`map`）以及自定义类型。

### 基本类型参数

基本类型参数是最常见的参数类型。例如：

```go
func greet(name string) {
    fmt.Println("Hello, " + name)
}
```

调用该函数时，传递一个字符串参数：

```go
greet("Alice") // 输出: Hello, Alice
```

### 复合类型参数

复合类型参数允许传递更复杂的数据结构。例如，传递一个整数切片：

```go
func sum(numbers []int) int {
    total := 0
    for _, num := range numbers {
        total += num
    }
    return total
}
```

调用该函数时，传递一个整数切片：

```go
numbers := []int{1, 2, 3, 4, 5}
result := sum(numbers)
fmt.Println(result) // 输出: 15
```

## 参数的传递方式

在Go语言中，函数参数的传递方式有两种：**值传递**和**引用传递**。

### 值传递

默认情况下，Go语言中的函数参数是值传递的。这意味着函数内部对参数的修改不会影响外部的原始值。

```go
func modifyValue(x int) {
    x = x + 10
    fmt.Println("Inside function:", x)
}

func main() {
    x := 5
    modifyValue(x)
    fmt.Println("Outside function:", x)
}
```

输出结果为：

```
Inside function: 15
Outside function: 5
```

可以看到，函数内部对 `x` 的修改并没有影响到外部的 `x`。

### 引用传递

如果需要修改外部的原始值，可以使用指针作为参数。指针参数允许函数直接操作原始数据。

```go
func modifyPointer(x *int) {
    *x = *x + 10
    fmt.Println("Inside function:", *x)
}

func main() {
    x := 5
    modifyPointer(&x)
    fmt.Println("Outside function:", x)
}
```

输出结果为：

```
Inside function: 15
Outside function: 15
```

通过传递指针，函数内部对 `x` 的修改影响到了外部的 `x`。

## 实际应用场景

### 案例1：计算多个数的平均值

假设我们需要编写一个函数来计算多个数的平均值。我们可以使用切片作为参数来接收这些数。

```go
func average(numbers []float64) float64 {
    total := 0.0
    for _, num := range numbers {
        total += num
    }
    return total / float64(len(numbers))
}

func main() {
    numbers := []float64{10.5, 20.3, 30.7, 40.2}
    avg := average(numbers)
    fmt.Println("Average:", avg) // 输出: Average: 25.425
}
```

### 案例2：交换两个变量的值

我们可以编写一个函数来交换两个变量的值。通过传递指针参数，我们可以直接修改原始变量的值。

```go
func swap(a *int, b *int) {
    temp := *a
    *a = *b
    *b = temp
}

func main() {
    x := 10
    y := 20
    swap(&x, &y)
    fmt.Println("x:", x, "y:", y) // 输出: x: 20 y: 10
}
```

## 总结

Go语言中的函数参数是函数与外部数据交互的重要方式。通过参数，我们可以将数据传递给函数，并在函数内部进行处理。Go语言支持值传递和引用传递两种参数传递方式，开发者可以根据需要选择合适的方式。

在实际开发中，函数参数的使用非常广泛。无论是处理基本类型还是复杂数据结构，函数参数都能帮助我们编写出更加灵活和可重用的代码。
