# C# 字符串搜索

在C#编程中，字符串搜索是一个常见的操作。无论是查找特定字符、子字符串，还是判断字符串中是否包含某些内容，掌握字符串搜索的方法都是非常重要的。本文将详细介绍C#中常用的字符串搜索方法，并通过代码示例和实际案例帮助你更好地理解这些概念。

## 1. 字符串搜索的基本概念

字符串搜索是指在给定的字符串中查找特定的字符或子字符串。C#提供了多种方法来实现这一功能，包括 `Contains`、`IndexOf`、`LastIndexOf`、`StartsWith` 和 `EndsWith` 等。这些方法可以帮助你快速定位字符串中的内容。

## 2. 常用的字符串搜索方法

### 2.1 `Contains` 方法

`Contains` 方法用于检查字符串中是否包含指定的子字符串。它返回一个布尔值，表示是否找到了匹配的内容。

```csharp
string text = "Hello, world!";
bool containsWorld = text.Contains("world");
Console.WriteLine(containsWorld); // 输出: True
```

### 2.2 `IndexOf` 方法

`IndexOf` 方法用于查找子字符串在字符串中首次出现的位置。如果找到匹配的内容，返回其索引；否则返回 `-1`。

```csharp
string text = "Hello, world!";
int index = text.IndexOf("world");
Console.WriteLine(index); // 输出: 7
```

### 2.3 `LastIndexOf` 方法

`LastIndexOf` 方法与 `IndexOf` 类似，但它查找的是子字符串在字符串中最后一次出现的位置。

```csharp
string text = "Hello, world! world!";
int lastIndex = text.LastIndexOf("world");
Console.WriteLine(lastIndex); // 输出: 14
```

### 2.4 `StartsWith` 方法

`StartsWith` 方法用于检查字符串是否以指定的子字符串开头。

```csharp
string text = "Hello, world!";
bool startsWithHello = text.StartsWith("Hello");
Console.WriteLine(startsWithHello); // 输出: True
```

### 2.5 `EndsWith` 方法

`EndsWith` 方法用于检查字符串是否以指定的子字符串结尾。

```csharp
string text = "Hello, world!";
bool endsWithWorld = text.EndsWith("world!");
Console.WriteLine(endsWithWorld); // 输出: True
```

## 3. 实际应用案例

### 3.1 检查用户输入

假设你正在开发一个应用程序，需要检查用户输入的电子邮件地址是否包含 `@` 符号。

```csharp
string email = "user@example.com";
if (email.Contains("@"))
{
    Console.WriteLine("Valid email address.");
}
else
{
    Console.WriteLine("Invalid email address.");
}
```

### 3.2 查找文件扩展名

在处理文件路径时，你可能需要检查文件的扩展名是否为 `.txt`。

```csharp
string filePath = "document.txt";
if (filePath.EndsWith(".txt"))
{
    Console.WriteLine("This is a text file.");
}
else
{
    Console.WriteLine("This is not a text file.");
}
```

## 4. 总结

在C#中，字符串搜索是一个基础但非常重要的操作。通过 `Contains`、`IndexOf`、`LastIndexOf`、`StartsWith` 和 `EndsWith` 等方法，你可以轻松地在字符串中查找特定的内容。掌握这些方法将帮助你在实际开发中更高效地处理字符串。
