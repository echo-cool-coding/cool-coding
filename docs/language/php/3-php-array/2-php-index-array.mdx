## 什么是PHP索引数组？

在PHP中，**索引数组**是一种以数字作为键名的数组类型。数组中的每个元素都有一个唯一的数字索引，从0开始递增。索引数组是PHP中最常用的数组类型之一，特别适合存储有序数据。

:::note
索引数组的键名是自动分配的，从0开始，依次递增。你也可以手动指定索引，但通常不建议这样做。
:::

## 创建索引数组

在PHP中，可以通过以下两种方式创建索引数组：

1. **使用`array()`函数**：
   ```php
   $fruits = array("Apple", "Banana", "Cherry");
   ```

2. **使用短数组语法`[]`**（PHP 5.4及以上版本）：
   ```php
   $fruits = ["Apple", "Banana", "Cherry"];
   ```

以上两种方式都会创建一个包含三个元素的索引数组，索引分别为0、1和2。

## 访问索引数组的元素

要访问索引数组中的元素，可以使用数组的索引值。例如：

```php
echo $fruits[0]; // 输出: Apple
echo $fruits[1]; // 输出: Banana
echo $fruits[2]; // 输出: Cherry
```

:::caution
如果尝试访问一个不存在的索引，PHP会抛出一个“未定义索引”的警告。例如：
```php
echo $fruits[3]; // 警告: Undefined offset: 3
```
:::

## 修改索引数组的元素

你可以通过索引直接修改数组中的元素：

```php
$fruits[1] = "Blueberry";
echo $fruits[1]; // 输出: Blueberry
```

## 遍历索引数组

遍历索引数组是常见的操作，可以使用`for`循环或`foreach`循环来实现。

### 使用`for`循环遍历

```php
for ($i = 0; $i < count($fruits); $i++) {
    echo $fruits[$i] . "<br />";
}
```

### 使用`foreach`循环遍历

```php
foreach ($fruits as $fruit) {
    echo $fruit . "<br />";
}
```

:::tip
`foreach`循环更适合遍历数组，因为它不需要手动管理索引。
:::

## 实际应用场景

索引数组在实际开发中有广泛的应用。以下是一些常见的场景：

1. **存储用户输入**：
   假设你有一个表单，用户输入了多个值，你可以将这些值存储在一个索引数组中：
   ```php
   $userInputs = ["John", "Doe", "john.doe@example.com"];
   ```

2. **处理数据库查询结果**：
   当你从数据库中查询数据时，结果通常以索引数组的形式返回：
   ```php
   $results = [
       ["id" => 1, "name" => "Alice"],
       ["id" => 2, "name" => "Bob"]
   ];
   ```

3. **生成动态HTML内容**：
   你可以使用索引数组动态生成HTML列表：
   ```php
   $items = ["Home", "About", "Contact"];
   echo "<ul>";
   foreach ($items as $item) {
       echo "<li>$item</li>";
   }
   echo "</ul>";
   ```
