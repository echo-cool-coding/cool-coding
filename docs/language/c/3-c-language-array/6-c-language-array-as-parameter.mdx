# C 语言数组作为参数

在C语言中，数组是一种非常重要的数据结构，它允许我们存储多个相同类型的元素。在实际编程中，我们经常需要将数组作为参数传递给函数，以便对数组进行操作或处理。本文将详细介绍如何在C语言中将数组作为参数传递给函数，并通过示例代码和实际案例帮助你理解这一概念。

## 1. 数组作为参数的基本概念

在C语言中，数组名本质上是一个指向数组第一个元素的指针。因此，当我们将数组作为参数传递给函数时，实际上传递的是数组的起始地址。这意味着函数内部可以通过指针来访问和修改数组中的元素。

:::note
**注意**：由于数组名是指针，因此在函数中修改数组元素会直接影响原始数组。
:::

## 2. 数组作为参数的语法

在C语言中，将数组作为参数传递给函数的语法如下：

```c
void functionName(int arr[], int size);
```

其中：
- `arr[]` 表示数组参数。
- `size` 表示数组的大小，通常需要显式传递数组的大小，因为数组名本身并不包含数组的长度信息。

## 3. 示例代码

下面是一个简单的示例，展示了如何将数组作为参数传递给函数，并在函数中打印数组的所有元素。

```c
#include <stdio.h>

// 函数声明
void printArray(int arr[], int size);

int main() {
    int myArray[] = {1, 2, 3, 4, 5};
    int size = sizeof(myArray) / sizeof(myArray[0]);

    // 调用函数并传递数组
    printArray(myArray, size);

    return 0;
}

// 函数定义
void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}
```

**输出**：
```
1 2 3 4 5
```

在这个示例中，`printArray` 函数接收一个整数数组和数组的大小作为参数，并打印数组的所有元素。

## 4. 数组作为参数的实际应用

在实际编程中，数组作为参数的应用非常广泛。例如，我们可以编写一个函数来计算数组的平均值：

```c
#include <stdio.h>

// 函数声明
double calculateAverage(int arr[], int size);

int main() {
    int myArray[] = {10, 20, 30, 40, 50};
    int size = sizeof(myArray) / sizeof(myArray[0]);

    // 调用函数并传递数组
    double average = calculateAverage(myArray, size);
    printf("Average: %.2f\n", average);

    return 0;
}

// 函数定义
double calculateAverage(int arr[], int size) {
    int sum = 0;
    for (int i = 0; i < size; i++) {
        sum += arr[i];
    }
    return (double)sum / size;
}
```

**输出**：
```
Average: 30.00
```

在这个示例中，`calculateAverage` 函数接收一个整数数组和数组的大小作为参数，并返回数组元素的平均值。

## 5. 多维数组作为参数

C语言还支持将多维数组作为参数传递给函数。例如，我们可以传递一个二维数组并计算其所有元素的总和：

```c
#include <stdio.h>

// 函数声明
int sum2DArray(int arr[][3], int rows);

int main() {
    int myArray[2][3] = {{1, 2, 3}, {4, 5, 6}};
    int rows = sizeof(myArray) / sizeof(myArray[0]);

    // 调用函数并传递二维数组
    int sum = sum2DArray(myArray, rows);
    printf("Sum: %d\n", sum);

    return 0;
}

// 函数定义
int sum2DArray(int arr[][3], int rows) {
    int sum = 0;
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < 3; j++) {
            sum += arr[i][j];
        }
    }
    return sum;
}
```

**输出**：
```
Sum: 21
```

在这个示例中，`sum2DArray` 函数接收一个二维数组和行数作为参数，并返回数组所有元素的总和。
