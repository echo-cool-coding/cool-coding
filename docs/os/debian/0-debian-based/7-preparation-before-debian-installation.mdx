# Debian 安装前准备

在开始安装Debian操作系统之前，做好充分的准备工作是至关重要的。这不仅可以帮助你避免安装过程中可能遇到的问题，还能确保系统安装后能够正常运行。本文将详细介绍在安装Debian之前需要完成的准备工作。

## 1. 硬件要求

首先，你需要确保你的计算机硬件满足Debian的最低要求。以下是Debian 11（Bullseye）的硬件要求：

- **处理器**：至少1 GHz的处理器。
- **内存**：至少512 MB的RAM（建议1 GB或更多）。
- **硬盘空间**：至少10 GB的可用空间（建议20 GB或更多）。
- **显卡**：支持1024x768分辨率的显卡。
- **网络连接**：虽然不是必须的，但建议有网络连接以便安装过程中下载更新和软件包。

:::tip
如果你的计算机硬件配置较低，可以考虑使用Debian的轻量级桌面环境（如LXDE或Xfce）以减少资源占用。
:::

## 2. 备份数据

在安装Debian之前，务必备份所有重要数据。安装过程中可能会涉及到磁盘分区和格式化操作，这可能会导致数据丢失。你可以使用外部硬盘、云存储或其他备份工具来保存你的数据。

:::caution
**重要提示**：即使你计划在双系统环境下安装Debian，也建议备份数据，以防意外情况发生。
:::

## 3. 下载Debian安装镜像

接下来，你需要从Debian官方网站下载适合你计算机架构的安装镜像。Debian支持多种架构，包括：

- **amd64**：适用于64位的Intel和AMD处理器。
- **i386**：适用于32位的Intel和AMD处理器。
- **arm64**：适用于64位的ARM处理器。

你可以访问[Debian官方网站](https://www.debian.org/distrib/)下载最新的安装镜像。

:::note
如果你不确定你的计算机架构，可以在Windows系统中通过以下步骤查看：
1. 打开“设置” > “系统” > “关于”。
2. 在“设备规格”下查看“系统类型”。
:::

## 4. 创建可启动的安装介质

下载完Debian安装镜像后，你需要将其写入USB闪存驱动器或DVD，以便从该介质启动并安装Debian。以下是使用Rufus工具在Windows系统中创建可启动USB驱动器的步骤：

1. 下载并安装[Rufus](https://rufus.ie/)。
2. 插入USB闪存驱动器。
3. 打开Rufus，选择你的USB驱动器。
4. 在“引导选择”中，点击“选择”按钮，选择你下载的Debian ISO文件。
5. 点击“开始”按钮，等待Rufus完成写入过程。

:::warning
**注意**：创建可启动介质会擦除USB驱动器上的所有数据，请确保已备份重要文件。
:::

## 5. 检查BIOS/UEFI设置

在启动安装介质之前，你需要确保计算机的BIOS/UEFI设置正确。以下是需要检查的几个关键设置：

- **启动顺序**：确保USB驱动器或DVD在启动顺序中位于首位。
- **安全启动**：如果你使用的是UEFI模式，可能需要禁用安全启动（Secure Boot）。
- **Legacy模式**：如果你使用的是较旧的计算机，可能需要启用Legacy模式。

:::tip
**提示**：进入BIOS/UEFI设置的方法因计算机品牌而异，通常在开机时按下`F2`、`F10`、`Del`或`Esc`键。
:::

## 6. 网络连接

虽然Debian可以在没有网络连接的情况下安装，但建议在安装过程中保持网络连接。这将允许你在安装过程中下载最新的软件包和更新，确保系统安装后是最新的。

:::note
如果你使用的是无线网络，可能需要提前准备好无线网络的SSID和密码。
:::

## 7. 分区规划

在安装Debian之前，建议你提前规划好磁盘分区。以下是一个常见的分区方案：

- **根分区（/）**：用于安装操作系统和应用程序，建议分配至少10 GB的空间。
- **交换分区（swap）**：用于虚拟内存，建议分配与物理内存相同或更大的空间。
- **家目录分区（/home）**：用于存储用户文件，建议分配剩余的空间。

:::caution
**注意**：如果你计划在双系统环境下安装Debian，请确保不要覆盖现有的操作系统分区。
:::

## 8. 总结

在安装Debian之前，完成以上准备工作可以大大减少安装过程中可能遇到的问题。确保你的硬件满足要求、备份重要数据、下载正确的安装镜像、创建可启动介质、检查BIOS/UEFI设置、保持网络连接，并提前规划好磁盘分区。

## 9. 附加资源与练习
