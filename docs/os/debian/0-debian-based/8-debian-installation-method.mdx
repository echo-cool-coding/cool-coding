## 介绍

Debian 是一个流行的 Linux 发行版，以其稳定性和广泛的软件包支持而闻名。无论你是想搭建服务器、开发环境，还是日常使用，Debian 都是一个不错的选择。本指南将带你一步步完成 Debian 的安装过程。

## 准备工作

在开始安装之前，你需要准备以下内容：

1. **下载 Debian 镜像**：访问 [Debian 官方网站](https://www.debian.org/distrib/) 下载最新的稳定版 ISO 文件。
2. **创建启动盘**：使用工具如 [Rufus](https://rufus.ie/) 或 [Etcher](https://www.balena.io/etcher/) 将 ISO 文件写入 U 盘。
3. **备份数据**：安装操作系统可能会格式化硬盘，请确保重要数据已备份。

## 安装步骤

### 1. 启动计算机并进入 BIOS/UEFI

将启动盘插入计算机后，重启并进入 BIOS/UEFI 设置界面。通常可以通过按下 `F2`、`F12`、`DEL` 或 `ESC` 键进入。在启动顺序中将 U 盘设置为第一启动项。

### 2. 启动 Debian 安装程序

保存 BIOS/UEFI 设置并重启计算机，系统将从 U 盘启动并进入 Debian 安装程序。

### 3. 选择安装模式

你会看到以下选项：

- **Graphical Install**：图形化安装界面，适合初学者。
- **Install**：文本模式安装，适合高级用户。

选择 `Graphical Install` 并按回车键。

### 4. 选择语言和区域设置

安装程序会提示你选择语言、地区和键盘布局。根据你的偏好进行选择。

### 5. 配置网络

安装程序会自动检测网络连接。如果连接成功，系统会提示你设置主机名和域名。

```bash
主机名: my-debian-pc
域名: example.com
```

### 6. 设置 root 密码

接下来，设置 root 用户的密码。请确保密码足够安全。

```bash
输入 root 密码: ********
确认密码: ********
```

### 7. 创建普通用户

为了安全起见，建议创建一个普通用户用于日常操作。

```bash
全名: John Doe
用户名: johndoe
密码: ********
```

### 8. 分区磁盘

Debian 提供了几种分区方式：

- **自动分区**：适合初学者，安装程序会自动为你分区。
- **手动分区**：适合高级用户，可以自定义分区方案。

选择 `自动分区` 并按提示操作。

### 9. 安装基本系统

安装程序会开始安装基本系统，这可能需要一些时间。

### 10. 选择软件包

安装完成后，系统会提示你选择要安装的软件包。你可以选择以下选项：

- **Debian 桌面环境**：如 GNOME、KDE 等。
- **Web 服务器**：如 Apache、Nginx。
- **SSH 服务器**：用于远程管理。

选择你需要的软件包并继续。

### 11. 安装 GRUB 引导加载程序

最后，安装程序会提示你安装 GRUB 引导加载程序。选择 `是` 并将其安装到硬盘的主引导记录（MBR）。

### 12. 完成安装

安装完成后，系统会提示你重启计算机。移除启动盘并按下回车键重启。

## 实际案例

假设你是一名开发者，需要在本地搭建一个开发环境。你可以选择安装 Debian 并配置以下软件包：

- **Git**：版本控制工具。
- **Python**：编程语言。
- **Docker**：容器化工具。

通过上述步骤，你可以轻松完成 Debian 的安装，并开始配置你的开发环境。
