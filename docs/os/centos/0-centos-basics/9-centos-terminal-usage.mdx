## 介绍

CentOS是一种广泛使用的Linux发行版，特别适合服务器环境。终端是CentOS中与系统交互的核心工具，通过命令行界面（CLI），用户可以执行各种任务，如文件管理、软件安装、系统监控等。对于初学者来说，掌握终端的基本使用是学习Linux的第一步。

## 终端的基本操作

### 打开终端

在CentOS中，你可以通过以下方式打开终端：

1. 使用快捷键 `Ctrl + Alt + T`。
2. 在桌面环境中，右键点击桌面并选择“打开终端”。
3. 在应用程序菜单中搜索“终端”并点击打开。

### 基本命令

以下是一些常用的终端命令：

- `pwd`：显示当前工作目录。
  ```bash
  $ pwd
  /home/username
  ```

- `ls`：列出当前目录下的文件和文件夹。
  ```bash
  $ ls
  Documents Downloads Music Pictures
  ```

- `cd`：切换目录。
  ```bash
  $ cd Documents
  $ pwd
  /home/username/Documents
  ```

- `mkdir`：创建新目录。
  ```bash
  $ mkdir new_folder
  $ ls
  Documents Downloads Music Pictures new_folder
  ```

- `rm`：删除文件或目录。
  ```bash
  $ rm file.txt
  $ rm -r folder
  ```

:::tip
使用 `rm -r` 时要小心，因为它会递归删除目录及其内容。
:::

### 文件操作

- `cp`：复制文件或目录。
  ```bash
  $ cp file.txt new_file.txt
  ```

- `mv`：移动或重命名文件或目录。
  ```bash
  $ mv file.txt new_location/file.txt
  $ mv old_name.txt new_name.txt
  ```

- `cat`：查看文件内容。
  ```bash
  $ cat file.txt
  This is the content of the file.
  ```

### 系统信息

- `uname`：显示系统信息。
  ```bash
  $ uname -a
  Linux hostname 3.10.0-1160.el7.x86_64 #1 SMP Mon Oct 19 16:18:59 UTC 2020 x86_64 x86_64 x86_64 GNU/Linux
  ```

- `top`：实时显示系统进程和资源使用情况。
  ```bash
  $ top
  ```

:::caution
`top` 命令会持续更新，按 `q` 退出。
:::

## 实际案例

### 案例1：安装软件

假设你想安装 `vim` 文本编辑器，可以使用以下命令：

```bash
$ sudo yum install vim
```

### 案例2：查找文件

如果你想查找名为 `example.txt` 的文件，可以使用 `find` 命令：

```bash
$ find / -name example.txt
```

### 案例3：查看日志文件

系统日志通常存储在 `/var/log` 目录下。你可以使用 `cat` 或 `tail` 命令查看日志文件：

```bash
$ tail -n 10 /var/log/messages
```
