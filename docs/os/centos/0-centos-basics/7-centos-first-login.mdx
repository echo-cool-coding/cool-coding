# CentOS 初次登录

## 介绍

CentOS是一种基于Linux的操作系统，广泛用于服务器环境。初次登录CentOS系统是学习Linux的第一步。本文将详细介绍如何通过命令行界面（CLI）登录CentOS系统，并完成一些基本的操作。

## 登录前的准备

在登录CentOS之前，确保你已经具备以下条件：

1. **系统安装**：CentOS已经成功安装在你的计算机或虚拟机上。
2. **网络连接**：确保系统已经连接到网络，以便进行后续的更新和配置。
3. **用户账户**：你已经创建了一个用户账户，或者知道root用户的密码。

## 登录步骤

### 1. 启动系统

启动你的CentOS系统。如果你使用的是虚拟机，确保虚拟机已经启动并加载了CentOS。

### 2. 进入登录界面

系统启动后，你会看到一个命令行登录界面。界面通常会显示如下内容：

```
CentOS Linux 7 (Core)
Kernel 3.10.0-1160.el7.x86_64 on an x86_64

localhost login:
```

### 3. 输入用户名

在`localhost login:`提示符后，输入你的用户名。如果你是第一次登录，可以使用`root`用户：

```
localhost login: root
```

### 4. 输入密码

输入用户名后，系统会提示你输入密码。输入密码时，屏幕上不会显示任何字符，这是正常的安全措施。

```
Password:
```

输入密码后，按`Enter`键。

### 5. 成功登录

如果用户名和密码正确，你将成功登录系统，并看到如下提示符：

```
[root@localhost ~]#
```

这表示你已经以`root`用户身份登录到系统。

## 基本命令行操作

登录成功后，你可以开始使用命令行进行操作。以下是一些基本的命令：

### 查看当前目录

使用`pwd`命令查看当前所在的目录：

```bash
[root@localhost ~]# pwd
/root
```

### 列出目录内容

使用`ls`命令列出当前目录下的文件和文件夹：

```bash
[root@localhost ~]# ls
anaconda-ks.cfg  Documents  Downloads  Music  Pictures  Public  Templates  Videos
```

### 创建新用户

为了安全起见，建议不要总是使用`root`用户进行操作。你可以创建一个普通用户：

```bash
[root@localhost ~]# adduser john
[root@localhost ~]# passwd john
```

系统会提示你为新用户设置密码。

### 切换用户

使用`su`命令切换到新创建的用户：

```bash
[root@localhost ~]# su - john
[john@localhost ~]$
```

## 实际应用场景

假设你是一名系统管理员，需要为新员工创建一个用户账户并设置初始密码。你可以按照以下步骤操作：

1. 使用`root`用户登录系统。
2. 使用`adduser`命令创建新用户。
3. 使用`passwd`命令为新用户设置密码。
4. 将新用户的账户信息告知员工。

## 总结

通过本文，你已经学会了如何在CentOS系统中完成初次登录，并掌握了一些基本的命令行操作。初次登录是使用CentOS的第一步，熟练掌握这些操作将为后续的学习打下坚实的基础。
