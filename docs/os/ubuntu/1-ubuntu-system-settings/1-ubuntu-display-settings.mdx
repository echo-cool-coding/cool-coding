# Ubuntu 显示设置

Ubuntu是一个功能强大的操作系统，广泛应用于开发、学习和日常使用。显示设置是系统配置中的重要部分，它直接影响用户的使用体验。本文将逐步讲解如何在Ubuntu中进行显示设置，包括分辨率调整、多显示器配置、亮度调节等。

## 1. 分辨率调整

分辨率决定了屏幕上显示内容的清晰度和大小。Ubuntu默认会根据显示器的支持情况自动设置分辨率，但有时你可能需要手动调整。

### 1.1 使用图形界面调整分辨率

1. 打开“设置”应用程序。
2. 选择“显示”选项。
3. 在“分辨率”下拉菜单中，选择适合你的显示器的分辨率。
4. 点击“应用”按钮保存更改。

### 1.2 使用命令行调整分辨率

如果你更喜欢使用命令行，可以使用`xrandr`命令来调整分辨率。

```bash
xrandr --output HDMI-1 --mode 1920x1080
```

- `HDMI-1`是你的显示器接口名称，可以通过`xrandr`命令查看。
- `1920x1080`是你想要设置的分辨率。

:::tip
你可以使用`xrandr`命令查看当前连接的显示器和支持的分辨率：
```bash
xrandr
```
:::

## 2. 多显示器配置

如果你有多个显示器，Ubuntu允许你轻松配置它们的工作方式，例如扩展显示或镜像显示。

### 2.1 使用图形界面配置多显示器

1. 打开“设置”应用程序。
2. 选择“显示”选项。
3. 你会看到所有连接的显示器。你可以拖动它们来调整它们的相对位置。
4. 选择“镜像显示”选项来复制屏幕内容，或者取消选择以扩展显示。

### 2.2 使用命令行配置多显示器

使用`xrandr`命令可以更灵活地配置多显示器。

```bash
xrandr --output HDMI-1 --auto --output HDMI-2 --auto --right-of HDMI-1
```

- `HDMI-1`和`HDMI-2`是你的显示器接口名称。
- `--right-of HDMI-1`表示将`HDMI-2`显示器放置在`HDMI-1`的右侧。

:::caution
确保你的显示器接口名称正确，否则命令可能无法生效。
:::

## 3. 亮度调节

亮度调节对于保护眼睛和节省电量非常重要。Ubuntu提供了多种方式来调节屏幕亮度。

### 3.1 使用图形界面调节亮度

1. 打开“设置”应用程序。
2. 选择“电源”选项。
3. 在“亮度”滑块中调整屏幕亮度。

### 3.2 使用命令行调节亮度

你可以使用`xbacklight`命令来调节亮度。

```bash
xbacklight -set 50
```

- `50`表示将亮度设置为50%。

:::note
如果`xbacklight`命令无效，可能需要安装相关软件包：
```bash
sudo apt install xbacklight
```
:::

## 4. 实际案例

### 4.1 案例1：调整分辨率以适应投影仪

假设你正在使用Ubuntu连接到投影仪进行演示，但投影仪的分辨率较低。你可以使用以下命令将分辨率调整为适合投影仪的分辨率：

```bash
xrandr --output VGA-1 --mode 1024x768
```

### 4.2 案例2：扩展显示以提高工作效率

如果你有一个外接显示器，你可以将其设置为扩展显示，以便在多个屏幕上同时工作。使用以下命令将外接显示器放置在主显示器的右侧：

```bash
xrandr --output HDMI-1 --auto --output HDMI-2 --auto --right-of HDMI-1
```
