# Ubuntu VPN 配置

VPN（虚拟专用网络）是一种通过加密和隧道技术在公共网络上创建安全连接的技术。它可以帮助您保护隐私、绕过地理限制或访问公司内部资源。在 Ubuntu 系统中，配置 VPN 是一个简单且实用的技能。本文将逐步指导您完成 Ubuntu 上的 VPN 配置。

---

## 什么是 VPN？

VPN 通过在您的设备和远程服务器之间建立加密连接，隐藏您的 IP 地址并保护您的网络流量。它通常用于以下场景：

- 访问受地理限制的内容（如流媒体服务）。
- 保护公共 Wi-Fi 上的隐私。
- 远程访问公司内部网络。

---

## 在 Ubuntu 中配置 VPN

Ubuntu 提供了图形化界面和命令行工具来配置 VPN。以下是两种方法的详细步骤。

### 方法 1：使用图形化界面配置 VPN

1. **打开网络设置**  
   点击右上角的网络图标，选择“设置” > “网络”。

2. **添加 VPN 连接**  
   在“网络”设置中，点击“VPN”选项卡，然后点击“添加 VPN”。

3. **选择 VPN 类型**  
   根据您的 VPN 提供商，选择合适的 VPN 类型（如 OpenVPN、PPTP 或 L2TP）。

4. **填写 VPN 配置信息**  
   输入 VPN 提供商提供的详细信息，包括网关、用户名、密码和证书（如果需要）。

5. **保存并连接**  
   点击“保存”，然后返回网络设置，选择新添加的 VPN 连接并点击“连接”。

:::tip
如果您不确定 VPN 类型或配置信息，请咨询您的 VPN 提供商。
:::

---

### 方法 2：使用命令行配置 VPN

如果您更喜欢使用命令行，可以通过 `nmcli` 工具配置 VPN。

1. **安装 VPN 客户端**  
   对于 OpenVPN，可以使用以下命令安装客户端：
   ```bash
   sudo apt update
   sudo apt install openvpn
   ```

2. **下载 VPN 配置文件**  
   从您的 VPN 提供商处下载配置文件（通常为 `.ovpn` 文件）。

3. **导入 VPN 配置**  
   使用以下命令导入配置文件：
   ```bash
   sudo nmcli connection import type openvpn file /path/to/your/config.ovpn
   ```

4. **配置 VPN 连接**  
   使用以下命令配置 VPN 连接：
   ```bash
   sudo nmcli connection modify "Your-VPN-Name" vpn.user-name "your-username"
   sudo nmcli connection modify "Your-VPN-Name" vpn.secrets "password=your-password"
   ```

5. **连接 VPN**  
   使用以下命令启动 VPN 连接：
   ```bash
   sudo nmcli connection up "Your-VPN-Name"
   ```

6. **验证连接**  
   使用以下命令检查 VPN 连接状态：
   ```bash
   nmcli connection show --active
   ```

:::caution
确保您的 VPN 配置文件路径和名称正确，否则连接会失败。
:::

## 实际案例：访问受地理限制的内容

假设您在国外旅行，希望访问某个仅限国内用户使用的流媒体服务。通过配置 VPN，您可以将流量路由到国内的服务器，从而绕过地理限制。

1. 选择一个支持国内服务器的 VPN 提供商。
2. 按照上述步骤配置 VPN。
3. 连接到国内服务器后，访问流媒体服务即可。
