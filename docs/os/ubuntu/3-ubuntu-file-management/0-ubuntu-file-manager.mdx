# Ubuntu 文件管理器

Ubuntu 文件管理器是 Ubuntu 操作系统中用于管理文件和文件夹的图形化工具。它提供了一个直观的界面，使用户可以轻松地浏览、复制、移动、删除和搜索文件。对于初学者来说，掌握文件管理器的基本操作是使用 Ubuntu 的第一步。

## 什么是 Ubuntu 文件管理器？

Ubuntu 文件管理器（也称为 **Nautilus** 或 **Files**）是 Ubuntu 默认的文件管理工具。它允许用户通过图形界面与文件系统进行交互，而无需使用命令行。文件管理器支持多种操作，如创建文件夹、重命名文件、查看文件属性等。

## 打开文件管理器

在 Ubuntu 中，你可以通过以下方式打开文件管理器：

1. 点击左侧 Dock 栏中的 **Files** 图标。
2. 按下 `Super` 键（通常是 Windows 键），然后搜索 **Files**。
3. 在终端中输入 `nautilus` 并按下回车键。

:::tip
如果你更喜欢使用命令行，可以通过终端直接打开文件管理器。例如，输入 `nautilus ~/Documents` 可以直接打开 Documents 文件夹。
:::

## 文件管理器的基本功能

### 1. 浏览文件和文件夹

文件管理器的主界面显示了当前目录下的所有文件和文件夹。你可以通过双击文件夹进入子目录，或双击文件打开它。

### 2. 创建文件夹

要创建一个新文件夹，可以右键点击空白处，然后选择 **New Folder**。你也可以使用快捷键 `Ctrl + Shift + N`。

### 3. 复制和移动文件

要复制文件，可以右键点击文件并选择 **Copy**，然后在目标文件夹中右键点击并选择 **Paste**。要移动文件，可以选择 **Cut** 而不是 **Copy**。

:::note
你也可以使用快捷键 `Ctrl + C` 复制文件，`Ctrl + X` 剪切文件，`Ctrl + V` 粘贴文件。
:::

### 4. 删除文件

要删除文件，可以右键点击文件并选择 **Move to Trash**。你也可以直接将文件拖到桌面上的 **Trash** 图标中。

:::caution
删除文件时，文件会被移动到垃圾桶中，而不是永久删除。要永久删除文件，你需要清空垃圾桶。
:::

### 5. 搜索文件

文件管理器右上角有一个搜索框。你可以在其中输入文件名或关键词来快速查找文件。

### 6. 查看文件属性

要查看文件的详细信息，可以右键点击文件并选择 **Properties**。这将显示文件的大小、类型、修改日期等信息。

## 实际案例

假设你正在整理你的 **Documents** 文件夹，以下是一个实际的操作流程：

1. 打开文件管理器并导航到 **Documents** 文件夹。
2. 创建一个名为 **Work** 的新文件夹。
3. 将所有与工作相关的文件移动到 **Work** 文件夹中。
4. 使用搜索功能查找所有 `.pdf` 文件，并将它们复制到 **Work** 文件夹中。
5. 删除不再需要的旧文件。

## 总结

Ubuntu 文件管理器是一个功能强大且易于使用的工具，适合初学者快速上手文件管理。通过掌握基本的操作，你可以轻松地组织和管理你的文件和文件夹。
