# Ubuntu 光盘刻录

在Ubuntu系统中，刻录光盘是一项常见的任务，无论是为了备份数据、创建安装介质，还是分发文件。本文将详细介绍如何在Ubuntu中刻录光盘，包括数据光盘和ISO镜像的刻录方法。

## 1. 准备工作

在开始刻录之前，确保你已经准备好以下内容：

- 一台支持光盘刻录的计算机。
- 一张空白的光盘（CD/DVD/Blu-ray）。
- 一个可用的光盘驱动器。
- Ubuntu系统已安装并运行。

## 2. 刻录数据光盘

### 2.1 使用Brasero刻录工具

Brasero是Ubuntu中一个简单易用的光盘刻录工具。如果你还没有安装Brasero，可以通过以下命令进行安装：

```bash
sudo apt-get install brasero
```

安装完成后，打开Brasero，选择“数据项目”选项。然后，将你想要刻录的文件拖放到Brasero窗口中。点击“刻录”按钮，选择光盘驱动器并开始刻录。

### 2.2 使用命令行刻录

如果你更喜欢使用命令行，可以使用`wodim`工具来刻录数据光盘。首先，确保你已经安装了`wodim`：

```bash
sudo apt-get install wodim
```

然后，使用以下命令刻录数据光盘：

```bash
wodim -v dev=/dev/sr0 -data /path/to/your/files
```

其中，`/dev/sr0`是你的光盘驱动器设备路径，`/path/to/your/files`是你要刻录的文件路径。

## 3. 刻录ISO镜像

### 3.1 使用Brasero刻录ISO镜像

打开Brasero，选择“刻录镜像”选项。然后，选择你要刻录的ISO文件，点击“刻录”按钮，选择光盘驱动器并开始刻录。

### 3.2 使用命令行刻录ISO镜像

同样地，你可以使用`wodim`工具来刻录ISO镜像。使用以下命令：

```bash
wodim -v dev=/dev/sr0 -dao /path/to/your/image.iso
```

其中，`/path/to/your/image.iso`是你要刻录的ISO文件路径。

## 4. 实际案例

### 4.1 刻录Ubuntu安装光盘

假设你下载了Ubuntu 20.04的ISO镜像，并希望将其刻录到光盘上以便安装。你可以按照以下步骤操作：

1. 打开Brasero，选择“刻录镜像”选项。
2. 选择你下载的Ubuntu 20.04 ISO文件。
3. 点击“刻录”按钮，选择光盘驱动器并开始刻录。

### 4.2 刻录数据备份光盘

假设你有一些重要的文档和图片需要备份到光盘上。你可以按照以下步骤操作：

1. 打开Brasero，选择“数据项目”选项。
2. 将你的文档和图片拖放到Brasero窗口中。
3. 点击“刻录”按钮，选择光盘驱动器并开始刻录。

## 5. 总结
