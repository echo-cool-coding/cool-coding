# Ubuntu 虚拟机软件

虚拟机软件允许你在现有的操作系统（如 Ubuntu）中运行另一个操作系统。这对于测试新软件、运行不同操作系统的应用程序或学习新系统非常有用。本文将介绍如何在 Ubuntu 上安装和使用虚拟机软件，并提供一个实际案例来帮助你理解其应用场景。

## 什么是虚拟机软件？

虚拟机软件（Virtual Machine Software）是一种允许你在现有操作系统上创建和运行虚拟计算机的软件。每个虚拟机都可以运行一个独立的操作系统，就像一台真实的计算机一样。常见的虚拟机软件包括 **VirtualBox** 和 **VMware**。

## 安装虚拟机软件

在 Ubuntu 上，最常用的虚拟机软件是 **VirtualBox**。以下是安装 VirtualBox 的步骤：

1. **更新软件包列表**  
   首先，确保你的系统是最新的：
   ```bash
   sudo apt update
   ```

2. **安装 VirtualBox**  
   使用以下命令安装 VirtualBox：
   ```bash
   sudo apt install virtualbox
   ```

3. **安装扩展包（可选）**  
   为了获得更多功能（如 USB 2.0/3.0 支持），你可以安装 VirtualBox 扩展包：
   ```bash
   sudo apt install virtualbox-ext-pack
   ```

4. **启动 VirtualBox**  
   安装完成后，你可以在应用程序菜单中找到 VirtualBox 并启动它。

## 创建虚拟机

安装完成后，你可以创建一个虚拟机来运行其他操作系统。以下是创建虚拟机的步骤：

1. **打开 VirtualBox**  
   启动 VirtualBox 后，点击“新建”按钮。

2. **设置虚拟机名称和类型**  
   输入虚拟机的名称，并选择操作系统的类型和版本。例如，如果你要安装 Windows 10，选择“Microsoft Windows”和“Windows 10 (64-bit)”。

3. **分配内存**  
   为虚拟机分配内存。建议至少分配 2GB 内存，但具体取决于你要运行的操作系统和应用程序。

4. **创建虚拟硬盘**  
   选择“现在创建虚拟硬盘”，然后选择“VDI (VirtualBox Disk Image)”作为硬盘文件类型。你可以选择“动态分配”或“固定大小”的硬盘。

5. **完成创建**  
   完成设置后，点击“创建”按钮。你的虚拟机现在已经创建好了。

## 安装操作系统

创建虚拟机后，你需要安装操作系统。以下是安装操作系统的步骤：

1. **加载安装介质**  
   在 VirtualBox 中，选择你创建的虚拟机，然后点击“设置”。在“存储”选项卡中，点击“空”光盘图标，然后选择“选择磁盘文件”来加载你的操作系统 ISO 文件。

2. **启动虚拟机**  
   启动虚拟机后，它将从 ISO 文件启动，并开始操作系统的安装过程。按照屏幕上的指示完成安装。

3. **安装 VirtualBox Guest Additions（可选）**  
   安装完成后，你可以安装 VirtualBox Guest Additions 以获得更好的性能和功能（如共享剪贴板和拖放文件）。

## 实际案例：在 Ubuntu 上运行 Windows 10

假设你需要在 Ubuntu 上运行一个只能在 Windows 上运行的应用程序。你可以按照以下步骤操作：

1. **创建 Windows 10 虚拟机**  
   按照上述步骤创建一个 Windows 10 虚拟机。

2. **安装 Windows 10**  
   使用 Windows 10 的 ISO 文件安装操作系统。

3. **运行应用程序**  
   安装完成后，你可以在虚拟机中运行 Windows 应用程序，就像在真实的 Windows 计算机上一样。

## 总结

虚拟机软件是 Ubuntu 用户运行其他操作系统的强大工具。通过 VirtualBox，你可以轻松创建和管理虚拟机，并在 Ubuntu 上运行 Windows、macOS 或其他 Linux 发行版。本文介绍了如何安装 VirtualBox、创建虚拟机以及安装操作系统的基本步骤。
