# Ubuntu 游戏支持

Ubuntu 是一个功能强大的开源操作系统，不仅适用于开发和生产环境，还逐渐成为游戏玩家的选择。虽然 Windows 长期以来一直是游戏的主要平台，但 Ubuntu 通过 Steam、Proton 和其他工具的支持，已经能够运行许多热门游戏。本文将带你了解如何在 Ubuntu 上安装和运行游戏，并探索 Ubuntu 对游戏的支持。

## 1. Ubuntu 上的游戏生态系统

Ubuntu 的游戏生态系统主要由以下几个部分组成：

- **Steam**：一个流行的游戏分发平台，支持 Linux 系统。
- **Proton**：由 Valve 开发的兼容层，允许 Windows 游戏在 Linux 上运行。
- **Lutris**：一个开源游戏平台，帮助用户轻松安装和管理游戏。
- **Wine**：一个兼容层，允许在 Linux 上运行 Windows 应用程序和游戏。

## 2. 安装 Steam

Steam 是 Ubuntu 上最受欢迎的游戏平台之一。以下是安装 Steam 的步骤：

1. 打开终端并更新软件包列表：
   ```bash
   sudo apt update
   ```

2. 安装 Steam：
   ```bash
   sudo apt install steam
   ```

3. 安装完成后，启动 Steam 并登录你的账户。

:::tip
如果你使用的是 Ubuntu 22.04 或更高版本，Steam 已经默认支持 64 位架构。
:::

## 3. 使用 Proton 运行 Windows 游戏

Proton 是 Valve 基于 Wine 开发的兼容层，允许你在 Ubuntu 上运行 Windows 游戏。以下是启用 Proton 的步骤：

1. 打开 Steam 并进入“设置”。
2. 在“Steam Play”选项卡中，勾选“为所有其他产品启用 Steam Play”。
3. 选择你希望使用的 Proton 版本（建议选择最新版本）。

现在，你可以像安装 Linux 游戏一样安装 Windows 游戏，Steam 会自动使用 Proton 来运行它们。

:::note
并非所有游戏都完全兼容 Proton，你可以在 [ProtonDB](https://www.protondb.com/) 上查看游戏的兼容性评级。
:::

## 4. 使用 Lutris 管理游戏

Lutris 是一个开源游戏平台，支持多种游戏来源（如 Steam、GOG、Epic Games Store 等）。以下是安装和使用 Lutris 的步骤：

1. 添加 Lutris 的 PPA 并安装：
   ```bash
   sudo add-apt-repository ppa:lutris-team/lutris
   sudo apt update
   sudo apt install lutris
   ```

2. 启动 Lutris 并添加你的游戏来源（如 Steam、GOG 等）。
3. 通过 Lutris 安装游戏，它会自动配置 Wine 或其他兼容层。

:::caution
Lutris 依赖于 Wine，因此某些游戏可能需要额外的配置才能正常运行。
:::

## 5. 实际案例：在 Ubuntu 上运行《Dota 2》

《Dota 2》是一款支持 Linux 的热门游戏。以下是安装和运行《Dota 2》的步骤：

1. 打开 Steam 并搜索《Dota 2》。
2. 点击“安装”并等待下载完成。
3. 安装完成后，点击“播放”按钮即可启动游戏。

:::tip
《Dota 2》是原生支持 Linux 的游戏之一，因此无需使用 Proton 或其他兼容层。
:::

## 6. 总结

Ubuntu 的游戏支持正在不断改进，通过 Steam、Proton 和 Lutris 等工具，你可以在 Ubuntu 上运行许多热门游戏。虽然并非所有游戏都能完美运行，但随着社区的不断努力，Ubuntu 正在成为一个越来越适合游戏的操作系统。
