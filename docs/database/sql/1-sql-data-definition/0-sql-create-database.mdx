# SQL 创建数据库

在 SQL 中，创建数据库是数据库管理的第一步。数据库是存储数据的容器，它可以包含多个表、视图、存储过程等对象。本文将详细介绍如何使用 SQL 语句创建数据库，并通过实际案例帮助你理解这一概念。

## 什么是数据库？

数据库是一个有组织的数据集合，通常以电子方式存储在计算机系统中。数据库管理系统（DBMS）用于管理数据库，而 SQL（结构化查询语言）是与数据库交互的标准语言。

## 创建数据库的基本语法

在 SQL 中，创建数据库的基本语法如下：

```sql
CREATE DATABASE database_name;
```

- `CREATE DATABASE` 是 SQL 语句的关键字，用于创建数据库。
- `database_name` 是你希望创建的数据库的名称。

:::note
数据库名称必须是唯一的，不能与现有数据库重名。
:::

### 示例：创建一个名为 `my_database` 的数据库

```sql
CREATE DATABASE my_database;
```

执行上述语句后，系统将创建一个名为 `my_database` 的数据库。

## 创建数据库时的可选参数

在某些数据库管理系统（如 MySQL、PostgreSQL）中，创建数据库时可以指定一些可选参数，例如字符集和排序规则。

### 示例：创建数据库并指定字符集和排序规则

```sql
CREATE DATABASE my_database
CHARACTER SET utf8mb4
COLLATE utf8mb4_unicode_ci;
```

- `CHARACTER SET utf8mb4` 指定数据库的字符集为 `utf8mb4`，这是一种支持更多字符的 Unicode 编码。
- `COLLATE utf8mb4_unicode_ci` 指定数据库的排序规则为 `utf8mb4_unicode_ci`，这是一种不区分大小写的排序规则。

:::tip
选择合适的字符集和排序规则可以确保数据库支持多语言数据，并满足特定的排序需求。
:::

## 实际应用场景

假设你正在开发一个多语言的电子商务网站，需要存储来自不同国家的用户数据。为了确保数据库能够正确处理各种语言的字符，你可以在创建数据库时指定 `utf8mb4` 字符集。

```sql
CREATE DATABASE ecommerce
CHARACTER SET utf8mb4
COLLATE utf8mb4_unicode_ci;
```

通过这种方式，你可以确保数据库能够存储和处理多种语言的文本数据。

## 总结

创建数据库是 SQL 数据定义的基础操作之一。通过 `CREATE DATABASE` 语句，你可以轻松创建一个新的数据库，并根据需要指定字符集和排序规则。掌握这一技能是进行数据库管理和开发的第一步。
