# MySQL 条件查询

在MySQL中，条件查询是数据库操作中最常用的功能之一。通过条件查询，我们可以从表中筛选出符合特定条件的数据。本文将详细介绍如何使用MySQL的`WHERE`子句进行条件查询，并通过实际案例帮助你更好地理解这一概念。

## 什么是条件查询？

条件查询是指在查询数据时，通过指定条件来筛选出符合要求的数据。在MySQL中，条件查询通常使用`WHERE`子句来实现。`WHERE`子句允许你指定一个或多个条件，只有满足这些条件的记录才会被返回。

## 基本语法

`WHERE`子句的基本语法如下：

```sql
SELECT 列名1, 列名2, ...
FROM 表名
WHERE 条件;
```

其中，`条件`可以是比较运算符、逻辑运算符或其他条件表达式。

## 常见的条件运算符

MySQL支持多种条件运算符，以下是一些常见的运算符：

- **比较运算符**：
  - `=`：等于
  - `<>` 或 `!=`：不等于
  - `>`：大于
  - `<`：小于
  - `>=`：大于等于
  - `<=`：小于等于

- **逻辑运算符**：
  - `AND`：与
  - `OR`：或
  - `NOT`：非

- **其他运算符**：
  - `BETWEEN`：在某个范围内
  - `LIKE`：模糊匹配
  - `IN`：在某个集合中
  - `IS NULL`：为空

## 示例1：使用比较运算符

假设我们有一个名为`students`的表，其中包含学生的`id`、`name`和`age`字段。我们想要查询年龄大于18岁的学生：

```sql
SELECT id, name, age
FROM students
WHERE age > 18;
```

**输出**：
```
+----+--------+-----+
| id | name   | age |
+----+--------+-----+
|  1 | Alice  |  20 |
|  2 | Bob    |  19 |
+----+--------+-----+
```

## 示例2：使用逻辑运算符

如果我们想要查询年龄大于18岁且名字为"Alice"的学生，可以使用`AND`运算符：

```sql
SELECT id, name, age
FROM students
WHERE age > 18 AND name = 'Alice';
```

**输出**：
```
+----+-------+-----+
| id | name  | age |
+----+-------+-----+
|  1 | Alice |  20 |
+----+-------+-----+
```

## 示例3：使用`BETWEEN`运算符

`BETWEEN`运算符用于查询某个范围内的数据。例如，查询年龄在18到20岁之间的学生：

```sql
SELECT id, name, age
FROM students
WHERE age BETWEEN 18 AND 20;
```

**输出**：
```
+----+--------+-----+
| id | name   | age |
+----+--------+-----+
|  1 | Alice  |  20 |
|  2 | Bob    |  19 |
+----+--------+-----+
```

## 示例4：使用`LIKE`运算符

`LIKE`运算符用于模糊匹配。例如，查询名字以"A"开头的学生：

```sql
SELECT id, name, age
FROM students
WHERE name LIKE 'A%';
```

**输出**：
```
+----+-------+-----+
| id | name  | age |
+----+-------+-----+
|  1 | Alice |  20 |
+----+-------+-----+
```

## 实际应用场景

假设你正在开发一个学生管理系统，你需要根据不同的条件查询学生信息。例如：

- 查询所有年龄大于18岁的学生。
- 查询名字以"A"开头且年龄在18到20岁之间的学生。
- 查询年龄为20岁或名字为"Bob"的学生。

这些查询都可以通过`WHERE`子句来实现。
