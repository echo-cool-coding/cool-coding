在编写HTML时，有时我们需要在网页中显示一些特殊字符，例如版权符号（`©`）、小于号（`<`）或大于号（`>`）。这些字符在HTML中有特殊的含义，如果直接输入，可能会导致页面渲染错误。为了避免这些问题，HTML提供了**特殊字符实体**来表示这些字符。

## 什么是HTML特殊字符实体？

HTML特殊字符实体是一种用于表示特殊字符的编码方式。它们以`&`开头，以`;`结尾。例如，`&lt;`表示小于号（`<`），`&gt;`表示大于号（`>`）。这些实体可以是**实体名称**（如`&lt;`）或**实体编号**（如`&#60;`）。

:::note
实体名称更易读，但并非所有特殊字符都有对应的实体名称。实体编号则适用于所有字符。
:::

## 常见的HTML特殊字符

以下是一些常见的HTML特殊字符及其对应的实体名称和实体编号：

| 字符 | 描述         | 实体名称   | 实体编号  |
|------|--------------|------------|-----------|
| `<`    | 小于号       | `&lt;`     | `&#60;`   |
|` >`    | 大于号       | `&gt;`     | `&#62;`   |
| `&`    | 和号         | `&amp;`    | `&#38;`   |
| `"`    | 双引号       | `&quot;`   | `&#34;`   |
| `'`    | 单引号       | `&apos;`   | `&#39;`   |
| `©`    | 版权符号     | `&copy;`   | `&#169;`  |
| `®`    | 注册商标符号 | `&reg;`    | `&#174;`  |
| `€`    | 欧元符号     | `&euro;`   | `&#8364;` |

## 如何在HTML中使用特殊字符

要在HTML中使用特殊字符，只需将对应的实体名称或实体编号插入到HTML代码中即可。例如：

```html
<p>这是一个小于号：&lt;</p>
<p>这是一个版权符号：&copy;</p>
<p>这是一个欧元符号：&euro;</p>
```

输出结果如下：

```
这是一个小于号：<
这是一个版权符号：©
这是一个欧元符号：€
```

:::tip
如果你不确定某个特殊字符的实体名称或编号，可以在网上搜索“HTML特殊字符表”来查找。
:::

## 实际应用场景

### 1. 显示数学公式

在网页中显示数学公式时，常常需要使用特殊字符。例如：

```html
<p>数学公式：x &lt; y &amp;&amp; y &gt; z</p>
```

输出结果：

```
数学公式：x < y && y > z
```

### 2. 显示版权信息

在网页底部显示版权信息时，通常会用到版权符号：

```html
<footer>&copy; 2023 我的网站</footer>
```

输出结果：

```
© 2023 我的网站
```

### 3. 显示货币符号

在电子商务网站中，显示价格时可能需要使用货币符号：

```html
<p>价格：&euro;19.99</p>
```

输出结果：

```
价格：€19.99
```

## 总结

HTML特殊字符实体是网页开发中不可或缺的一部分。通过使用实体名称或实体编号，我们可以确保特殊字符在网页中正确显示，而不会干扰HTML的结构。无论是显示数学符号、版权信息还是货币符号，掌握这些特殊字符的使用方法都将使你的网页更加专业和准确。
