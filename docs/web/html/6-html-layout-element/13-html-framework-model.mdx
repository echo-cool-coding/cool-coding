# HTML 框架模型

HTML框架模型是网页布局的基础，它定义了网页中各个元素的排列方式和相互关系。理解框架模型对于构建结构清晰、易于维护的网页至关重要。本文将逐步介绍HTML框架模型的核心概念，并通过实际案例帮助你掌握其应用。

## 什么是HTML框架模型？

HTML框架模型是指网页中元素的布局方式，包括块级元素（block-level elements）和行内元素（inline elements）。块级元素通常用于构建页面的主要结构，而行内元素则用于处理文本和其他内容。

### 块级元素 vs 行内元素

- **块级元素**：块级元素会占据其父元素的整个宽度，并在前后添加换行符。常见的块级元素包括 `<div>`、`<p>`、`<h1>` 到 `<h6>` 等。
- **行内元素**：行内元素只占据其内容所需的宽度，不会强制换行。常见的行内元素包括 `<span>`、`<a>`、`<strong>` 等。

:::tip
块级元素可以包含其他块级元素和行内元素，而行内元素通常只能包含其他行内元素。
:::

## HTML 布局元素

HTML提供了一系列用于布局的元素，这些元素可以帮助你构建复杂的网页结构。以下是一些常用的布局元素：

### `<div>` 元素

`<div>` 是最常用的块级元素之一，通常用于将内容分组或创建布局容器。

```html
<div>
  <p>这是一个段落。</p>
  <p>这是另一个段落。</p>
</div>
```

### `<span>` 元素

`<span>` 是一个行内元素，通常用于对文本的一部分进行样式化或添加事件处理程序。

```html
<p>这是一个 <span style="color: red;">红色</span> 的文本。</p>
```

### `<header>`、`<footer>`、`<section>` 等语义化元素

HTML5引入了许多语义化元素，这些元素不仅有助于构建页面结构，还能提高代码的可读性和可维护性。

```html
<header>
  <h1>网站标题</h1>
</header>
<section>
  <h2>文章标题</h2>
  <p>文章内容...</p>
</section>
<footer>
  <p>版权信息</p>
</footer>
```

## 实际案例：构建一个简单的网页布局

让我们通过一个简单的案例来展示如何使用HTML框架模型构建网页布局。

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>简单网页布局</title>
</head>
<body>
  <header>
    <h1>欢迎来到我的网站</h1>
    <nav>
      <ul>
        <li><a href="#">首页</a></li>
        <li><a href="#">关于我们</a></li>
        <li><a href="#">联系我们</a></li>
      </ul>
    </nav>
  </header>
  <main>
    <section>
      <h2>最新文章</h2>
      <article>
        <h3>文章标题</h3>
        <p>文章内容...</p>
      </article>
    </section>
  </main>
  <footer>
    <p>&copy; 2023 我的网站</p>
  </footer>
</body>
</html>
```

### 输出结果

上述代码将生成一个简单的网页布局，包含页眉、导航栏、主要内容区域和页脚。

## 总结

HTML框架模型是网页布局的基础，理解块级元素和行内元素的区别以及如何使用语义化元素是构建结构清晰、易于维护的网页的关键。通过本文的学习，你应该能够使用HTML布局元素构建简单的网页结构。
