# HTML 响应式表格

在现代网页设计中，响应式设计是一个至关重要的概念。响应式表格是指能够根据用户设备的屏幕尺寸自动调整布局和样式的表格。这对于移动设备用户尤其重要，因为传统的表格在小屏幕上可能会显得拥挤或难以阅读。

## 什么是响应式表格？

响应式表格是一种能够根据屏幕宽度自动调整布局的表格。当屏幕宽度较小时，表格可能会重新排列列，或者通过滚动条来显示所有内容。这样可以确保用户在任何设备上都能轻松查看和理解表格数据。

## 如何创建响应式表格

### 1. 使用CSS媒体查询

CSS媒体查询是创建响应式表格的基础。通过媒体查询，我们可以根据屏幕宽度应用不同的样式。

```css
/* 默认样式 */
table {
  width: 100%;
  border-collapse: collapse;
}

/* 当屏幕宽度小于600px时应用的样式 */
@media screen and (max-width: 600px) {
  table, thead, tbody, th, td, tr {
    display: block;
  }
  th {
    position: absolute;
    top: -9999px;
    left: -9999px;
  }
  tr {
    border: 1px solid #ccc;
  }
  td {
    border: none;
    border-bottom: 1px solid #eee;
    position: relative;
    padding-left: 50%;
  }
  td:before {
    position: absolute;
    top: 6px;
    left: 6px;
    width: 45%;
    padding-right: 10px;
    white-space: nowrap;
    content: attr(data-label);
  }
}
```

### 2. HTML结构

在HTML中，我们需要为每个表格单元格添加 `data-label` 属性，以便在小屏幕上显示列标题。

```html
<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Age</th>
      <th>Occupation</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td data-label="Name">John Doe</td>
      <td data-label="Age">28</td>
      <td data-label="Occupation">Web Developer</td>
    </tr>
    <tr>
      <td data-label="Name">Jane Smith</td>
      <td data-label="Age">34</td>
      <td data-label="Occupation">Graphic Designer</td>
    </tr>
  </tbody>
</table>
```

### 3. 实际效果

在大屏幕上，表格会正常显示所有列。而在小屏幕上，表格会重新排列，每一行变成一个块级元素，并且每个单元格前面会显示对应的列标题。

:::note
**注意**：在实际开发中，你可能需要根据具体需求调整CSS样式，以确保表格在不同设备上都能良好显示。
:::

## 实际案例

假设你正在开发一个电商网站，需要在产品页面上显示一个包含产品名称、价格和库存的表格。为了确保在移动设备上也能良好显示，你可以使用上述方法创建一个响应式表格。

```html
<table>
  <thead>
    <tr>
      <th>Product Name</th>
      <th>Price</th>
      <th>Stock</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td data-label="Product Name">Laptop</td>
      <td data-label="Price">$1200</td>
      <td data-label="Stock">10</td>
    </tr>
    <tr>
      <td data-label="Product Name">Smartphone</td>
      <td data-label="Price">$800</td>
      <td data-label="Stock">25</td>
    </tr>
  </tbody>
</table>
```

## 总结

响应式表格是现代网页设计中不可或缺的一部分。通过使用CSS媒体查询和适当的HTML结构，我们可以确保表格在不同设备上都能良好显示。这对于提升用户体验至关重要。
