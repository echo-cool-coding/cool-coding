# CSS 旋转变换

CSS旋转变换是一种强大的工具，允许开发者通过简单的代码旋转网页中的元素。无论是创建动态效果还是调整布局，旋转变换都能为你的设计增添灵活性。本文将详细介绍CSS旋转变换的语法、使用方法以及实际应用场景。

## 什么是CSS旋转变换？

CSS旋转变换是CSS3中引入的一种变换类型，允许你围绕一个固定点旋转元素。旋转的角度可以是正值（顺时针）或负值（逆时针）。通过旋转变换，你可以轻松地改变元素的方向，而无需修改其内容或结构。

## 基本语法

CSS旋转变换的基本语法如下：

```css
transform: rotate(角度);
```

其中，`角度`可以是度数（deg）、弧度（rad）、梯度（grad）或圈数（turn）。最常用的是度数（deg）。

### 示例：旋转45度

```css
div {
  transform: rotate(45deg);
}
```

在这个例子中，`div`元素将顺时针旋转45度。

## 旋转中心点

默认情况下，元素的旋转中心是其中心点。你可以通过`transform-origin`属性来改变旋转中心点。

### 示例：改变旋转中心点

```css
div {
  transform: rotate(45deg);
  transform-origin: top left;
}
```

在这个例子中，`div`元素将以左上角为旋转中心点旋转45度。

## 实际应用场景

### 1. 创建旋转图标

旋转变换常用于创建动态图标，如加载动画或旋转箭头。

```css
.loading-icon {
  animation: spin 2s linear infinite;
}

@keyframes spin {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}
```

在这个例子中，`.loading-icon`元素将无限旋转，模拟加载动画。

### 2. 调整元素方向

旋转变换也可以用于调整元素的方向，例如将文本旋转90度以创建垂直标题。

```css
.vertical-text {
  transform: rotate(90deg);
  transform-origin: left top;
}
```

在这个例子中，`.vertical-text`元素将以左上角为旋转中心点旋转90度，使文本垂直显示。

## 总结

CSS旋转变换是一种简单而强大的工具，可以帮助你轻松地旋转网页元素。通过掌握`rotate()`函数和`transform-origin`属性，你可以创建各种动态效果和布局调整。希望本文能帮助你更好地理解和应用CSS旋转变换。
