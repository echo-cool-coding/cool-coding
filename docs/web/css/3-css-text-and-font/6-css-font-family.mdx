## 介绍

在网页设计中，字体的选择对用户体验至关重要。CSS字体系列（`font-family`）属性允许开发者指定一个或多个字体名称，浏览器会按照顺序尝试加载这些字体。如果第一个字体不可用，浏览器会尝试下一个字体，直到找到可用的字体为止。这种机制确保了文本在不同设备和浏览器上都能正确显示。

## 基本语法

`font-family`属性的基本语法如下：

```css
selector {
  font-family: "字体名称", 备用字体, 通用字体系列;
}
```

- **字体名称**：指定首选字体，通常用引号包裹。
- **备用字体**：如果首选字体不可用，浏览器会尝试加载备用字体。
- **通用字体系列**：如果所有指定字体都不可用，浏览器会使用通用字体系列中的字体。

### 通用字体系列

CSS定义了五种通用字体系列：

1. **serif**：衬线字体，如Times New Roman。
2. **sans-serif**：无衬线字体，如Arial。
3. **monospace**：等宽字体，如Courier New。
4. **cursive**：手写体，如Comic Sans MS。
5. **fantasy**：装饰性字体，如Impact。

## 代码示例

以下是一个简单的示例，展示了如何使用`font-family`属性：

```css
body {
  font-family: "Georgia", "Times New Roman", serif;
}
```

在这个例子中，浏览器会首先尝试加载Georgia字体。如果Georgia不可用，它会尝试加载Times New Roman。如果两者都不可用，浏览器会使用serif通用字体系列中的字体。

## 实际应用场景

### 1. 网页正文

对于网页正文，通常选择易读的衬线或无衬线字体。例如：

```css
article {
  font-family: "Arial", "Helvetica", sans-serif;
}
```

### 2. 代码块

对于代码块，通常使用等宽字体以确保代码对齐：

```css
pre {
  font-family: "Courier New", "Courier", monospace;
}
```

### 3. 标题

对于标题，可以选择更具装饰性的字体以吸引注意力：

```css
h1 {
  font-family: "Impact", "Arial Black", sans-serif;
}
```

## 注意事项

- **字体可用性**：并非所有用户的设备上都安装了指定的字体。因此，建议使用Web安全字体或通过`@font-face`加载自定义字体。
- **性能**：加载过多自定义字体可能会影响网页性能，因此应谨慎使用。

## 总结

`font-family`属性是CSS中用于控制文本字体的重要工具。通过合理选择字体和备用字体，可以确保网页在不同设备和浏览器上都能保持良好的可读性和美观性。
