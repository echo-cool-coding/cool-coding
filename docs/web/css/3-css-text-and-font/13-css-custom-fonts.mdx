# CSS 自定义字体

在网页设计中，字体是传达信息和风格的重要元素。虽然现代浏览器支持多种系统字体，但有时我们需要使用自定义字体来实现独特的设计效果。CSS提供了`@font-face`规则，允许开发者加载并使用自定义字体。本文将详细介绍如何使用CSS自定义字体，并提供实际案例帮助你掌握这一技术。

## 什么是CSS自定义字体？

CSS自定义字体是指通过CSS加载并使用非系统默认字体的技术。通过`@font-face`规则，开发者可以指定字体的名称、来源文件以及格式，从而在网页中使用这些字体。这种方式使得网页设计更加灵活，能够实现更丰富的视觉效果。

## 使用@font-face规则

`@font-face`是CSS中用于定义自定义字体的规则。它的基本语法如下：

```css
@font-face {
  font-family: 'MyCustomFont';
  src: url('my-custom-font.woff2') format('woff2'),
       url('my-custom-font.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}
```

### 参数解释

- `font-family`: 定义字体的名称，可以在CSS中通过这个名称引用字体。
- `src`: 指定字体文件的路径和格式。可以指定多个来源，浏览器会按顺序尝试加载。
- `font-weight`: 定义字体的粗细，例如`normal`、`bold`等。
- `font-style`: 定义字体的样式，例如`normal`、`italic`等。

### 字体格式

常见的字体格式包括：

- **WOFF/WOFF2**: 现代浏览器广泛支持的字体格式，压缩率高，加载速度快。
- **TTF/OTF**: 传统字体格式，兼容性较好，但文件较大。
- **EOT**: 主要用于旧版IE浏览器。

:::tip
建议优先使用WOFF2格式，因为它具有更好的压缩率和加载性能。同时，提供WOFF格式作为备选，以确保兼容性。
:::

## 实际应用案例

假设我们有一个名为`MyCustomFont`的字体文件，我们希望在整个网站中使用这个字体。以下是实现步骤：

1. **定义字体**：在CSS中使用`@font-face`规则定义字体。

```css
@font-face {
  font-family: 'MyCustomFont';
  src: url('fonts/my-custom-font.woff2') format('woff2'),
       url('fonts/my-custom-font.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}
```

2. **应用字体**：在CSS中通过`font-family`属性引用自定义字体。

```css
body {
  font-family: 'MyCustomFont', sans-serif;
}
```

在这个例子中，如果`MyCustomFont`无法加载，浏览器会回退到`sans-serif`字体。

## 真实应用场景

### 品牌一致性

许多品牌有自己独特的字体，用于保持视觉一致性。通过自定义字体，可以在网站上使用与品牌一致的字体，增强品牌识别度。

### 特殊设计需求

在某些设计中，可能需要使用非常规字体来实现特定的视觉效果。例如，复古风格的设计可能需要使用手写体或装饰性字体。

## 总结

CSS自定义字体为网页设计提供了更多的灵活性和创意空间。通过`@font-face`规则，开发者可以轻松加载并使用自定义字体，实现独特的视觉效果。在实际应用中，建议优先使用WOFF2格式，并提供备选格式以确保兼容性。
