# CSS 元素选择器

CSS（层叠样式表）是用于描述HTML文档外观的语言。CSS选择器是CSS的核心部分，它允许我们选择特定的HTML元素并为其应用样式。**CSS元素选择器**是最基本的选择器之一，它通过HTML元素的名称来选择元素。

## 什么是CSS元素选择器？

CSS元素选择器（也称为类型选择器）通过HTML元素的名称来选择元素。例如，如果你想为所有的段落（`<p>`）元素设置样式，你可以使用元素选择器 `p`。

### 语法

```css
元素名称 {
  属性: 值;
}
```

- **元素名称**：你想要选择的HTML元素的名称，例如 `p`、`h1`、`div` 等。
- **属性**：你想要设置的CSS属性，例如 `color`、`font-size`、`background-color` 等。
- **值**：属性的具体值，例如 `red`、`16px`、`#ffffff` 等。

### 示例

假设我们有以下HTML代码：

```html
<p>这是一个段落。</p>
<p>这是另一个段落。</p>
<h1>这是一个标题</h1>
```

我们可以使用CSS元素选择器为所有的段落设置红色文本颜色：

```css
p {
  color: red;
}
```

应用上述CSS后，所有的段落文本将变为红色，而标题文本将保持不变。

## 实际应用场景

### 1. 统一页面样式

元素选择器常用于为页面中的特定元素设置统一的样式。例如，你可能希望所有的标题（`<h1>`、`<h2>` 等）使用相同的字体和颜色：

```css
h1, h2, h3 {
  font-family: Arial, sans-serif;
  color: #333;
}
```

### 2. 重置默认样式

浏览器通常会为某些HTML元素（如 `<body>`、`<p>`、`<h1>` 等）应用默认样式。你可以使用元素选择器来重置这些默认样式：

```css
body {
  margin: 0;
  padding: 0;
  font-family: Arial, sans-serif;
}
```

### 3. 响应式设计

在响应式设计中，元素选择器可以用于为不同屏幕尺寸设置不同的样式。例如，你可以为小屏幕设备设置较小的字体大小：

```css
p {
  font-size: 16px;
}

@media (max-width: 600px) {
  p {
    font-size: 14px;
  }
}
```

## 总结

CSS元素选择器是CSS中最基本且常用的选择器之一。它通过HTML元素的名称来选择元素，并为其应用样式。通过掌握元素选择器，你可以轻松地为页面中的特定元素设置统一的样式，重置默认样式，甚至实现响应式设计。

:::tip 提示
虽然元素选择器非常强大，但在实际开发中，通常会结合其他选择器（如类选择器、ID选择器）来更精确地选择元素并应用样式。
:::
