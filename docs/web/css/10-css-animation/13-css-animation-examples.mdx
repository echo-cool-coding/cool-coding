# CSS 动画实例

CSS动画是一种强大的工具，可以让网页元素动起来，从而增强用户体验。通过定义关键帧（`@keyframes`）和使用动画属性（如 `animation`），你可以创建各种动态效果。本文将带你逐步学习CSS动画的基础知识，并通过实际案例展示其应用。

## 什么是CSS动画？

CSS动画允许你在不依赖JavaScript的情况下，通过CSS代码实现元素的动态变化。它通过定义关键帧（`@keyframes`）来描述动画的各个阶段，然后通过 `animation` 属性将动画应用到元素上。

### 关键帧（`@keyframes`）

关键帧是CSS动画的核心。它定义了动画在不同时间点的状态。例如，你可以定义一个从0%到100%的动画，分别描述元素在动画开始、中间和结束时的样式。

```css
@keyframes example {
  0%   { background-color: red; }
  50%  { background-color: yellow; }
  100% { background-color: green; }
}
```

### 动画属性（`animation`）

`animation` 属性用于将关键帧动画应用到元素上。它包含多个子属性，如 `animation-name`、`animation-duration`、`animation-timing-function` 等。

```css
div {
  animation-name: example;
  animation-duration: 4s;
  animation-timing-function: ease-in-out;
  animation-iteration-count: infinite;
}
```

## 基本动画示例

让我们从一个简单的动画开始。假设我们想让一个方块在页面上从左到右移动。

```css
@keyframes moveRight {
  0%   { transform: translateX(0); }
  100% { transform: translateX(200px); }
}

.box {
  width: 100px;
  height: 100px;
  background-color: blue;
  animation: moveRight 2s ease-in-out infinite;
}
```

在这个例子中，`moveRight` 动画将方块从初始位置向右移动200像素，持续时间为2秒，并且动画会无限循环。

## 实际应用场景

### 1. 按钮悬停效果

按钮悬停效果是CSS动画的常见应用之一。通过添加简单的动画，可以让按钮在用户悬停时产生视觉反馈。

```css
@keyframes hoverEffect {
  0%   { transform: scale(1); }
  50%  { transform: scale(1.1); }
  100% { transform: scale(1); }
}

.button {
  padding: 10px 20px;
  background-color: #007BFF;
  color: white;
  border: none;
  cursor: pointer;
  animation: hoverEffect 0.5s ease-in-out;
}

.button:hover {
  animation: hoverEffect 0.5s ease-in-out;
}
```

在这个例子中，当用户将鼠标悬停在按钮上时，按钮会稍微放大，然后恢复到原始大小。

### 2. 加载动画

加载动画是另一个常见的应用场景。通过CSS动画，你可以创建一个旋转的加载图标。

```css
@keyframes spin {
  0%   { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

.loader {
  width: 50px;
  height: 50px;
  border: 5px solid #f3f3f3;
  border-top: 5px solid #3498db;
  border-radius: 50%;
  animation: spin 1s linear infinite;
}
```

这个例子中，`spin` 动画会让加载图标无限旋转，形成一个常见的加载效果。

## 总结

CSS动画是网页设计中不可或缺的一部分，它可以让你的网页更加生动和有趣。通过定义关键帧和使用动画属性，你可以轻松创建各种动态效果。本文通过几个简单的例子展示了CSS动画的基本用法，希望你能通过这些实例掌握CSS动画的基础知识。
