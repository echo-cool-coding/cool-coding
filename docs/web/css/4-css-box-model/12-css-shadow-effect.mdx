# CSS 阴影效果

在网页设计中，阴影效果是一种常见的视觉增强技术，能够为元素添加深度和层次感。CSS提供了两种主要的阴影效果：`box-shadow` 和 `text-shadow`。本文将详细介绍这两种属性的使用方法，并通过实际案例展示它们的应用场景。

## 1. `box-shadow` 属性

`box-shadow` 属性用于为元素的边框添加阴影效果。它可以控制阴影的颜色、大小、模糊度、偏移量以及是否内嵌。

### 语法

```css
box-shadow: h-offset v-offset blur spread color inset;
```

- **h-offset**: 水平偏移量，正值表示阴影向右偏移，负值表示向左偏移。
- **v-offset**: 垂直偏移量，正值表示阴影向下偏移，负值表示向上偏移。
- **blur**: 模糊半径，值越大，阴影越模糊。
- **spread**: 阴影的扩展半径，正值会使阴影扩大，负值会使阴影缩小。
- **color**: 阴影的颜色，可以使用任何有效的CSS颜色值。
- **inset**: 可选参数，如果指定，阴影将内嵌在元素内部。

### 示例

```css
.box {
  width: 200px;
  height: 200px;
  background-color: #f0f0f0;
  box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.5);
}
```

**输出效果：**

<div style={{
    width: '200px',
    height: '200px',
    backgroundColor: '#f0f0f0',
    boxShadow: '10px 10px 20px rgba(0, 0, 0, 0.5)'
}}></div>

:::tip
你可以通过调整 `h-offset` 和 `v-offset` 的值来改变阴影的方向，通过调整 `blur` 和 `spread` 的值来改变阴影的模糊程度和大小。
:::

## 2. `text-shadow` 属性

`text-shadow` 属性用于为文本添加阴影效果。它可以控制阴影的颜色、大小、模糊度和偏移量。

### 语法

```css
text-shadow: h-offset v-offset blur color;
```

- **h-offset**: 水平偏移量，正值表示阴影向右偏移，负值表示向左偏移。
- **v-offset**: 垂直偏移量，正值表示阴影向下偏移，负值表示向上偏移。
- **blur**: 模糊半径，值越大，阴影越模糊。
- **color**: 阴影的颜色，可以使用任何有效的CSS颜色值。

### 示例

```css
.text {
  font-size: 24px;
  color: #333;
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
}
```

**输出效果：**

<div style={{ fontSize: '24px', color: '#333', textShadow: '2px 2px 4px rgba(0, 0, 0, 0.5)' }}>Hello, World!</div>

:::note
`text-shadow` 可以叠加多个阴影效果，只需用逗号分隔不同的阴影参数即可。
:::

## 3. 实际应用场景

### 3.1 卡片设计

在卡片设计中，`box-shadow` 常用于为卡片添加立体感。例如：

```css
.card {
  width: 300px;
  padding: 20px;
  background-color: #fff;
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
}
```

**输出效果：**

<div style={{ width: '300px', padding: '20px', backgroundColor: '#fff', boxShadow: '0 4px 8px rgba(0, 0, 0, 0.1)' }}>这是一个卡片示例</div>

### 3.2 文字突出显示

在标题或重要文本中，`text-shadow` 可以用于突出显示文字。例如：

```css
.title {
  font-size: 36px;
  color: #fff;
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.7);
}
```

**输出效果：**

<div style={{ fontSize: '36px', color: '#fff', textShadow: '2px 2px 4px rgba(0, 0, 0, 0.7)' }}>突出显示的标题</div>

## 4. 总结

CSS阴影效果是增强网页视觉层次感的重要工具。通过 `box-shadow` 和 `text-shadow` 属性，你可以轻松地为元素和文本添加阴影效果，从而提升页面的美观度和用户体验。

:::caution
在使用阴影效果时，注意不要过度使用，以免影响页面的可读性和加载性能。
:::
