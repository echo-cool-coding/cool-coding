# Arduino 原理图绘制

## 介绍

Arduino是一款开源的电子原型平台，广泛应用于各种电子项目中。要理解Arduino的工作原理，首先需要掌握如何绘制其原理图。原理图是电路设计的蓝图，它展示了电路中各个组件的连接方式。通过绘制原理图，你可以清晰地了解电路的运行逻辑，并为后续的硬件搭建打下基础。

## 什么是原理图？

原理图是一种用符号表示电路连接关系的图形。它使用标准化的符号来表示电子元件（如电阻、电容、晶体管等），并通过线条表示这些元件之间的连接。原理图不涉及元件的物理布局，而是专注于电路的逻辑结构。

## 绘制原理图的基本步骤

### 1. 选择绘图工具
绘制原理图的第一步是选择合适的工具。常用的工具包括：
- **Fritzing**：适合初学者，界面友好。
- **KiCad**：功能强大，适合复杂的电路设计。
- **Eagle**：广泛应用于专业电路设计。

### 2. 了解元件符号
在绘制原理图之前，你需要熟悉常见的电子元件符号。以下是一些常见的元件符号：
- **电阻**：用锯齿线表示。
- **电容**：用两条平行线表示。
- **LED**：用箭头和圆圈表示。
- **Arduino**：通常用一个矩形表示，标注引脚名称。

### 3. 绘制电路连接
使用绘图工具将元件符号放置在画布上，并用线条连接它们。确保连接线清晰、准确，避免交叉和混淆。

### 4. 标注元件参数
在原理图中标注元件的参数（如电阻值、电容值等），以便后续的硬件搭建。

### 5. 检查电路逻辑
完成原理图后，仔细检查电路的逻辑是否正确，确保没有短路或断路。

## 实际案例：LED闪烁电路

让我们通过一个简单的LED闪烁电路来实践原理图绘制。

### 电路描述
该电路使用Arduino控制一个LED的闪烁。LED通过一个220Ω的电阻连接到Arduino的数字引脚。

### 原理图绘制步骤

1. **放置Arduino符号**：在画布上放置Arduino符号，并标注引脚名称（如D13、GND）。
2. **放置LED符号**：将LED符号放置在Arduino符号旁边。
3. **放置电阻符号**：在LED和Arduino之间放置一个220Ω的电阻符号。
4. **连接电路**：用线条将Arduino的D13引脚连接到电阻，电阻连接到LED的正极，LED的负极连接到Arduino的GND引脚。
5. **标注参数**：在电阻符号旁边标注“220Ω”。

### 代码示例

```cpp
void setup() {
  pinMode(13, OUTPUT); // 设置D13引脚为输出模式
}

void loop() {
  digitalWrite(13, HIGH); // 点亮LED
  delay(1000); // 等待1秒
  digitalWrite(13, LOW); // 熄灭LED
  delay(1000); // 等待1秒
}
```

### 输入与输出
- **输入**：无外部输入。
- **输出**：LED每隔1秒闪烁一次。
