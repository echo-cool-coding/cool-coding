# Arduino 智能农业

## 介绍

智能农业是利用现代技术（如物联网、传感器和自动化）来提高农业生产效率的一种方法。Arduino作为一种开源硬件平台，因其易用性和灵活性，成为构建智能农业系统的理想选择。通过Arduino，我们可以监测土壤湿度、温度、光照等环境参数，并自动控制灌溉、通风等设备，从而实现精准农业。

本文将逐步介绍如何使用Arduino构建一个简单的智能农业系统，涵盖传感器数据采集、自动化控制以及实际应用案例。

## 所需硬件

- Arduino Uno
- 土壤湿度传感器
- DHT11温湿度传感器
- 光敏电阻（LDR）
- 水泵模块
- 继电器模块
- 面包板和跳线

## 步骤1：连接传感器

首先，将传感器连接到Arduino。以下是连接方式：

- **土壤湿度传感器**：VCC -> 5V, GND -> GND, A0 -> A0
- **DHT11温湿度传感器**：VCC -> 5V, GND -> GND, Data -> D2
- **光敏电阻**：一端连接到5V，另一端通过10kΩ电阻连接到GND，中间节点连接到A1
- **水泵模块**：VCC -> 5V, GND -> GND, IN -> D3

:::note
确保所有连接正确无误，避免短路或损坏设备。
:::

## 步骤2：编写代码

接下来，编写Arduino代码来读取传感器数据并控制水泵。以下是一个简单的示例代码：

```cpp
#include <DHT.h>

#define DHTPIN 2
#define DHTTYPE DHT11
#define SOIL_MOISTURE_PIN A0
#define LIGHT_SENSOR_PIN A1
#define PUMP_PIN 3

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
  pinMode(PUMP_PIN, OUTPUT);
}

void loop() {
  float humidity = dht.readHumidity();
  float temperature = dht.readTemperature();
  int soilMoisture = analogRead(SOIL_MOISTURE_PIN);
  int lightLevel = analogRead(LIGHT_SENSOR_PIN);

  Serial.print("Humidity: ");
  Serial.print(humidity);
  Serial.print(" %, Temperature: ");
  Serial.print(temperature);
  Serial.print(" °C, Soil Moisture: ");
  Serial.print(soilMoisture);
  Serial.print(", Light Level: ");
  Serial.println(lightLevel);

  if (soilMoisture < 500) { // 假设500为干燥阈值
    digitalWrite(PUMP_PIN, HIGH); // 打开水泵
  } else {
    digitalWrite(PUMP_PIN, LOW); // 关闭水泵
  }

  delay(2000); // 每2秒读取一次数据
}
```

:::tip
你可以根据实际需求调整阈值和控制逻辑。例如，添加光照条件来控制水泵。
:::

## 步骤3：测试系统

上传代码后，打开串口监视器，观察传感器数据。如果土壤湿度低于设定阈值，水泵将自动启动。通过调整阈值和逻辑，你可以优化系统的响应。

## 实际应用案例

### 案例1：自动化灌溉系统

在一个小型农场中，农民使用Arduino构建了一个自动化灌溉系统。系统根据土壤湿度和天气预报自动调整灌溉计划，节省了水资源并提高了作物产量。

### 案例2：温室环境监控

在一个温室中，Arduino被用来监测温度、湿度和光照强度。当环境参数超出设定范围时，系统会自动启动通风设备或遮阳网，确保作物生长在最佳环境中。

## 总结
