# 路由器配置基础

## 介绍

路由器是网络中的关键设备，负责在不同网络之间转发数据包。通过配置路由器，您可以控制数据包的传输路径，确保网络的高效运行。本文将介绍路由器配置的基础知识，帮助您理解如何配置路由器以满足网络需求。

## 路由器配置的基本概念

在开始配置路由器之前，您需要了解一些基本概念：

- **接口（Interface）**：路由器的物理或逻辑端口，用于连接不同的网络。
- **IP 地址（IP Address）**：每个接口都需要一个唯一的 IP 地址，以便在网络中进行通信。
- **路由表（Routing Table）**：路由器使用路由表来决定如何转发数据包。
- **静态路由（Static Routing）**：手动配置的路由条目，适用于小型网络。
- **动态路由（Dynamic Routing）**：通过路由协议自动学习和更新路由表，适用于大型网络。

## 路由器配置步骤

### 1. 访问路由器

首先，您需要通过控制台、Telnet 或 SSH 访问路由器的命令行界面（CLI）。以下是使用 SSH 访问路由器的示例：

```bash
ssh admin@192.168.1.1
```

输入密码后，您将进入路由器的 CLI。

### 2. 配置接口

接下来，您需要为路由器的接口配置 IP 地址。以下是一个配置示例：

```bash
Router> enable
Router# configure terminal
Router(config)# interface GigabitEthernet0/0
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
```

在这个示例中，我们为 `GigabitEthernet0/0` 接口配置了 IP 地址 `192.168.1.1` 和子网掩码 `255.255.255.0`，并启用了该接口。

### 3. 配置静态路由

如果您需要手动配置路由，可以使用静态路由。以下是一个配置示例：

```bash
Router(config)# ip route 192.168.2.0 255.255.255.0 192.168.1.2
```

在这个示例中，我们配置了一条静态路由，将所有发往 `192.168.2.0/24` 网络的数据包通过 `192.168.1.2` 转发。

### 4. 保存配置

完成配置后，您需要将配置保存到路由器的非易失性存储器中，以便在路由器重启后仍然有效。以下是保存配置的命令：

```bash
Router# write memory
```

## 实际应用场景

假设您有一个小型办公室网络，包含两个子网：`192.168.1.0/24` 和 `192.168.2.0/24`。您需要配置路由器，使这两个子网能够互相通信。

1. **配置接口**：为路由器的两个接口分别配置 IP 地址。
2. **配置静态路由**：确保路由器知道如何转发数据包。
3. **测试连接**：使用 `ping` 命令测试两个子网之间的连通性。

以下是一个完整的配置示例：

```bash
Router> enable
Router# configure terminal
Router(config)# interface GigabitEthernet0/0
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
Router(config)# interface GigabitEthernet0/1
Router(config-if)# ip address 192.168.2.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
Router(config)# ip route 192.168.2.0 255.255.255.0 192.168.1.2
Router(config)# exit
Router# write memory
```
