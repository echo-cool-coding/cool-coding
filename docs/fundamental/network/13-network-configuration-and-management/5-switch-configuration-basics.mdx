## 介绍

交换机是计算机网络中的关键设备，用于在局域网（LAN）中连接多台设备并转发数据包。通过配置交换机，您可以优化网络性能、提高安全性并实现更高效的网络管理。本文将介绍交换机的基础配置，帮助您快速上手。

:::note
交换机配置通常通过命令行界面（CLI）完成，因此熟悉基本命令是学习的第一步。
:::

## 交换机的基本配置

### 1. 访问交换机

大多数交换机支持通过以下方式访问：
- **控制台连接**：使用串口线连接交换机的控制台端口。
- **远程访问**：通过 SSH 或 Telnet 远程登录交换机。

#### 示例：通过控制台连接
```bash
# 使用串口工具（如 PuTTY）连接交换机
# 配置串口参数：波特率 9600，数据位 8，无奇偶校验，停止位 1
```

#### 示例：通过 SSH 远程登录
```bash
# 使用 SSH 客户端登录交换机
ssh admin@192.168.1.1
```

---

### 2. 基本命令

以下是一些常用的交换机配置命令：

#### 查看交换机信息
```bash
# 查看交换机的基本信息
show version
```

#### 配置主机名
```bash
# 将交换机的主机名设置为 "Switch1"
configure terminal
hostname Switch1
exit
```

#### 保存配置
```bash
# 将当前配置保存到启动配置中
write memory
```

---

### 3. VLAN 配置

VLAN（虚拟局域网）用于将网络划分为多个逻辑子网，以提高安全性和管理效率。

#### 创建 VLAN
```bash
# 创建 VLAN 10 并命名为 "Sales"
configure terminal
vlan 10
name Sales
exit
```

#### 将端口分配到 VLAN
```bash
# 将端口 1 分配到 VLAN 10
interface fastethernet 0/1
switchport mode access
switchport access vlan 10
exit
```

---

### 4. 端口管理

交换机端口可以配置为访问模式（Access）或中继模式（Trunk）。

#### 配置访问模式
```bash
# 将端口 2 配置为访问模式，并分配到 VLAN 20
interface fastethernet 0/2
switchport mode access
switchport access vlan 20
exit
```

#### 配置中继模式
```bash
# 将端口 3 配置为中继模式，允许所有 VLAN 通过
interface fastethernet 0/3
switchport mode trunk
switchport trunk allowed vlan all
exit
```

## 实际案例

### 场景：小型办公室网络配置

假设您需要为一个小型办公室配置交换机，要求如下：
1. 创建两个 VLAN：`Sales`（VLAN 10）和 `IT`（VLAN 20）。
2. 将端口 1-5 分配给 `Sales`，端口 6-10 分配给 `IT`。
3. 配置端口 11 为中继端口，连接到路由器。

#### 配置步骤
```bash
# 创建 VLAN
configure terminal
vlan 10
name Sales
exit
vlan 20
name IT
exit

# 分配端口到 VLAN
interface range fastethernet 0/1-5
switchport mode access
switchport access vlan 10
exit

interface range fastethernet 0/6-10
switchport mode access
switchport access vlan 20
exit

# 配置中继端口
interface fastethernet 0/11
switchport mode trunk
switchport trunk allowed vlan all
exit
```
