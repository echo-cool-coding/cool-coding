# 无线局域网

无线局域网（Wireless Local Area Network，简称 WLAN）是一种通过无线信号连接设备的技术，允许设备在没有物理电缆的情况下进行通信。它是现代网络技术的重要组成部分，广泛应用于家庭、办公室、公共场所等场景。

## 什么是无线局域网？

无线局域网是一种基于无线电波或红外线的网络技术，允许设备在有限范围内进行无线通信。与有线局域网（LAN）不同，WLAN 不需要物理电缆连接设备，而是通过无线接入点（Access Point，AP）和无线网卡实现设备之间的通信。

### 无线局域网的主要特点
- **灵活性**：设备可以在覆盖范围内自由移动。
- **易于扩展**：无需布线即可增加新设备。
- **成本效益**：减少布线成本，特别适用于难以布线的环境。

---

## 无线局域网的工作原理

无线局域网的核心是无线接入点（AP），它充当有线网络和无线设备之间的桥梁。AP 通过无线电波与设备通信，并将数据转发到有线网络或其他无线设备。

### 无线局域网的组成
1. **无线接入点（AP）**：负责发送和接收无线信号。
2. **无线客户端**：如笔记本电脑、智能手机等支持无线网络的设备。
3. **无线网卡**：安装在设备中，用于接收和发送无线信号。
4. **路由器**：通常与 AP 集成，用于连接互联网。

## 无线局域网的技术标准

无线局域网的技术标准由 IEEE 802.11 系列协议定义。以下是常见的 WLAN 标准：

| 标准       | 频率范围   | 最大速率 | 特点                         |
|------------|------------|----------|------------------------------|
| 802.11a    | 5 GHz      | 54 Mbps  | 高速，但覆盖范围较小         |
| 802.11b    | 2.4 GHz    | 11 Mbps  | 低速，但覆盖范围较大         |
| 802.11g    | 2.4 GHz    | 54 Mbps  | 兼容 802.11b，速度更快       |
| 802.11n    | 2.4/5 GHz  | 600 Mbps | 支持多输入多输出（MIMO）技术 |
| 802.11ac   | 5 GHz      | 1.3 Gbps | 更高速度，支持更多设备       |
| 802.11ax   | 2.4/5 GHz  | 10 Gbps  | 高效能，适用于高密度环境     |

:::tip
802.11ax（Wi-Fi 6）是目前最新的标准，提供了更高的速度和更好的性能，特别适合现代高密度网络环境。
:::

## 无线局域网的实际应用

无线局域网广泛应用于以下场景：

### 1. 家庭网络
家庭中的无线局域网通常由无线路由器提供，支持多个设备同时连接互联网，如智能手机、平板电脑、智能电视等。

### 2. 办公网络
办公室中的无线局域网允许员工在办公区域内自由移动，同时保持网络连接。它还支持会议室、休息区等场所的网络覆盖。

### 3. 公共场所
咖啡馆、机场、图书馆等公共场所通常提供免费的无线局域网服务，方便用户访问互联网。

### 4. 工业环境
在工厂或仓库中，无线局域网用于连接移动设备（如手持扫描仪）和中央管理系统，提高工作效率。

## 无线局域网的安全性

无线局域网的安全性是一个重要问题，因为无线信号容易被窃听或攻击。以下是常见的无线局域网安全措施：

1. **WEP（有线等效加密）**：早期的加密技术，安全性较低，已逐渐被淘汰。
2. **WPA（Wi-Fi 保护访问）**：比 WEP 更安全，支持动态密钥。
3. **WPA2/WPA3**：目前最常用的加密标准，提供更高的安全性。

:::caution
使用弱密码或未加密的无线网络可能导致数据泄露或网络攻击。务必启用 WPA2 或 WPA3 加密。
:::
