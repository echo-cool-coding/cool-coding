# Lean 复数

复数是数学中的一个重要概念，广泛应用于工程、物理和计算机科学等领域。在 Lean 中，复数可以通过标准库中的定义进行操作。本文将介绍如何在 Lean 中定义复数、进行基本运算，并通过实际案例展示其应用。

## 什么是复数？

复数是由实数部分和虚数部分组成的数，通常表示为 `a + bi`，其中 `a` 是实部，`b` 是虚部，`i` 是虚数单位，满足 `i² = -1`。复数在解决某些实数范围内无解的问题时非常有用，例如求解某些多项式方程。

## 在 Lean 中定义复数

在 Lean 中，复数可以通过 `ℂ` 类型来表示。我们可以使用 `complex` 模块来定义和操作复数。以下是一个简单的例子，展示如何定义一个复数：

```lean
import data.complex.basic

-- 定义一个复数
def my_complex : ℂ := ⟨3, 4⟩

#check my_complex  -- 输出: my_complex : ℂ
#eval my_complex   -- 输出: 3 + 4i
```

在上面的代码中，我们定义了一个复数 `my_complex`，其实部为 `3`，虚部为 `4`。通过 `#check` 和 `#eval` 命令，我们可以分别查看该复数的类型和值。

## 复数的基本运算

在 Lean 中，我们可以对复数进行加法、减法、乘法和除法等基本运算。以下是一些示例：

```lean
import data.complex.basic

-- 定义两个复数
def z1 : ℂ := ⟨2, 3⟩
def z2 : ℂ := ⟨4, 5⟩

-- 复数加法
def z_add := z1 + z2
#eval z_add  -- 输出: 6 + 8i

-- 复数减法
def z_sub := z1 - z2
#eval z_sub  -- 输出: -2 - 2i

-- 复数乘法
def z_mul := z1 * z2
#eval z_mul  -- 输出: -7 + 22i

-- 复数除法
def z_div := z1 / z2
#eval z_div  -- 输出: 0.5609756097560976 + 0.0487804878048781i
```

:::note
复数的除法运算涉及到复数的共轭和模的计算，因此结果可能看起来比较复杂。
:::

## 复数的共轭和模

复数的共轭和模是复数的重要属性。共轭复数是将虚部取负的复数，而模是复数在复平面上的距离。以下是如何在 Lean 中计算复数的共轭和模：

```lean
import data.complex.basic

-- 定义一个复数
def z : ℂ := ⟨3, 4⟩

-- 计算共轭复数
def z_conj := z.conj
#eval z_conj  -- 输出: 3 - 4i

-- 计算复数的模
def z_abs := z.abs
#eval z_abs  -- 输出: 5
```

:::tip
复数的模可以通过勾股定理计算，即 `|a + bi| = √(a² + b²)`。
:::

## 实际应用案例

复数在信号处理、量子力学和电路分析等领域有广泛应用。以下是一个简单的电路分析案例，展示如何使用复数计算交流电路中的阻抗：

```lean
import data.complex.basic

-- 定义电阻和电感的阻抗
def R : ℝ := 5  -- 电阻
def L : ℝ := 2   -- 电感
def ω : ℝ := 10  -- 角频率

-- 计算电感的阻抗
def Z_L : ℂ := ⟨0, ω * L⟩

-- 计算总阻抗
def Z_total : ℂ := ⟨R, 0⟩ + Z_L
#eval Z_total  -- 输出: 5 + 20i
```

在这个案例中，我们计算了一个包含电阻和电感的交流电路的总阻抗。通过使用复数，我们可以轻松地将电阻和电感的阻抗结合起来。
